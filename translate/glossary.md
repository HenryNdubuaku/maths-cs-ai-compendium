# Глоссарий терминов (English → Русский)

Этот файл передаётся модели, чтобы термины переводились единообразно во всех главах.
Редактируйте свободно: строки формата `| English | Русский |` подхватываются автоматически
(строки-заголовки и разделители таблицы игнорируются). Можно также писать `English -> Русский`.

Правила: при первом появлении узкоспециального термина допустимо дать английский вариант
в скобках, напр. «внимание (attention)». Устоявшиеся заимствования не переводим силой
(«эмбеддинг», «токен», «пайплайн»), если русский аналог звучит хуже.
