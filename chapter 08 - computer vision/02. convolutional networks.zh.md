# 卷积网络

*卷积神经网络直接从像素数据中学习空间特征层级，用梯度优化的滤波器替代手工设计的滤波器。本文件涵盖卷积机制、池化、步长、空洞、感受野以及定义了图像分类的里程碑架构（LeNet、AlexNet、VGG、ResNet、Inception、EfficientNet）。*

- 在文件 01 中，我们手工设计了用于边缘检测、模糊和角点检测的滤波器。自然的问题是：我们能从数据中学习最优滤波器吗？这正是卷积神经网络（CNN）所做的。

- CNN 不再手工选择滤波器权重，而是通过梯度下降（第 06 章）学习它们，发现对当前任务直接有用的特征。

- 在第 06 章中，我们介绍了卷积操作、CNN 基础以及滤波器学习的思想。这里我们深入探讨使 CNN 在十多年间成为计算机视觉主导范式的架构创新。

- 回顾核心**卷积操作**：一个大小为 $k \times k$ 的滤波器 $K$ 在输入 feature map 上滑动，在每个位置计算点积（第 06 章）。输出大小由三个超参数控制：

    - **步长（stride）**：滤波器在位置之间移动多少像素。步长 1 表示滤波器每次移动一个像素。步长 2 表示移动两个像素，使空间维度减半。带步长的卷积是池化进行下采样的替代方案。
    - **填充（padding）**：在输入边界周围补零。"Same" 填充（$p = \lfloor k/2 \rfloor$）保持空间维度。"Valid" 填充（$p = 0$）会减小维度。
    - **空洞（dilation）**：在滤波器元素之间插入间隔。一个 dilation 为 2 的 3x3 滤波器只使用 9 个参数就覆盖了 5x5 的感受野。空洞卷积在不增加计算量的情况下扩大感受野。

- 卷积后的输出空间大小：

$$\text{out} = \left\lfloor \frac{\text{in} - k + 2p}{s} \right\rfloor + 1$$

- 其中 $\text{in}$ 是输入大小，$k$ 是 kernel 大小，$p$ 是 padding，$s$ 是 stride。该公式独立地应用于高度和宽度。

- 神经元的**感受野**（receptive field）是指原始输入中能影响其值的区域。
    - 浅层有较小的感受野（它们看到局部模式如边缘）。
    - 深层有较大的感受野（它们看到更大的结构如物体部件）。

- 感受野随每层增长：每个卷积层大约增加 $k - 1$ 个像素（有 stride 或 dilation 时更多）。

![感受野随层增长：第 1 层神经元看到 3x3 块，第 2 层神经元看到 5x5 块，第 3 层神经元看到原始输入的 7x7 块](../images/receptive_field.svg)

- **池化**（pooling）层在保留最重要信息的同时减小空间维度。
    - **最大池化**（max pooling）取每个窗口中的最大值，保留最强的激活（最显著的特征）。
    - **平均池化**（average pooling）取均值，平滑 feature map。步长为 2 的 2x2 池化将两个空间维度都减半。

- **全局平均池化**（Global Average Pooling，GAP）将每个通道的整个空间范围平均为一个数值，产生一个长度等于通道数的向量。GAP 在许多现代架构的末端替代了全连接层，大幅减少了参数量并起到结构性正则化的作用。

- **批量归一化**（Batch Normalization，BatchNorm）将每个小批量内的激活归一化为零均值和单位方差，然后应用一个可学习的缩放和平移（第 06 章）。在 CNN 中，BatchNorm 按通道应用：对每个通道独立地在批量和空间维度上计算统计量。它稳定训练、允许更高的学习率，并起到轻度正则化的作用。

- **Dropout**（第 06 章）在训练期间随机将神经元置零。

- 在 CNN 中，**空间 dropout**（Dropout2D）丢弃整个 feature map 通道而不是单个像素，这更有效，因为 feature map 中相邻像素高度相关。

- **数据增强**通过在训练期间对每张图像应用随机变换来人工扩展训练集：水平翻转、随机裁剪、旋转、颜色抖动（调整亮度、对比度、饱和度、色相）以及 cutout（遮盖随机矩形块）。网络以多种不同形式看到每张图像，迫使其学习变换不变的特征，而非记忆特定像素模式。

- 高级增强策略包括 **Mixup**（混合两张图像及其标签：$\tilde{x} = \lambda x_i + (1-\lambda) x_j$，$\tilde{y} = \lambda y_i + (1-\lambda) y_j$）、**CutMix**（将一张图像的矩形块粘贴到另一张图像上，并按面积比例混合标签）以及 **RandAugment**（从固定集合中以单一强度参数随机采样一系列增强操作）。

- CNN 架构的历史是逐步变得更深、更高效的设计故事，每一个都解决了限制其前代的问题。

- **LeNet-5**（LeCun 等，1998）是最初的 CNN，为手写数字识别而设计。两个卷积层后接三个全连接层，使用平均池化和 tanh 激活。它证明了学习到的滤波器优于手工设计的特征，但按现代标准它很小（60K 参数）。

- **AlexNet**（Krizhevsky 等，2012）以巨大优势赢得 ImageNet 竞赛，点燃了深度学习革命。关键创新：ReLU 激活（替代 tanh，后者受梯度消失困扰）、用于正则化的 dropout、数据增强以及在 GPU 上训练。五个卷积层、三个全连接层、6000 万参数。

- **VGG**（Simonyan 和 Zisserman，2014）表明仅使用 3x3 滤波器深度堆叠效果优于更大的滤波器。两个堆叠的 3x3 滤波器与一个 5x5 滤波器有相同的感受野，但参数更少（$2 \times 3^2 = 18$ vs $5^2 = 25$）且多一层非线性。VGG-16（16 层）和 VGG-19（19 层）至今仍被广泛用作特征提取器。其架构非常简单：通道数递增（64、128、256、512）的卷积块，每块后接最大池化。

![VGG 架构：堆叠的 3x3 卷积块，通道深度递增（64→128→256→512），块之间有最大池化，以全连接层结束](../images/vgg_architecture.svg)

- **GoogLeNet/Inception**（Szegedy 等，2014）引入了 **Inception 模块**：不再选择单一滤波器大小，而是并行使用 1x1、3x3 和 5x5 卷积，将它们的输出拼接，让网络决定哪个尺度最有用。1x1 卷积在更大的滤波器之前用作瓶颈以减少计算量。GoogLeNet 以比 VGG 少 12 倍的参数（6.8M vs 138M）取得了更好的精度。

![Inception 模块：四条并行分支（1×1、3×3、5×5 和池化），带 1×1 瓶颈，沿通道维度拼接](../images/inception_module.svg)

- Inception 模块同时捕获多尺度特征。1x1 滤波器捕获点状模式，3x3 捕获局部纹理，5x5 捕获更大结构。拼接将所有视角组合成一个丰富的表示。

- **ResNet**（He 等，2016）解决了**退化问题**：更深的网络表现比浅层更差，不是因为过拟合，而是因为它们更难优化。解决方案是**跳跃连接**（skip connection，残差连接）：

$$\text{output} = F(x) + x$$

- 该层学习残差 $F(x) = \text{output} - x$。如果最优变换接近恒等（在深度网络中很常见），学习一个接近零的残差比学习完整映射容易得多。跳跃连接还提供了直接的梯度高速通道，缓解梯度消失。ResNet 训练了 152 层的网络，远比以往任何网络都深。

![ResNet 块：输入 x 经过两个卷积层产生 F(x)，然后跳跃连接把 x 加回，得到输出 F(x) + x](../images/resnet_block.svg)

- 当输入和输出维度不同（由于 stride 或通道变化）时，**投影捷径**对 $x$ 应用一个 1x1 卷积以匹配维度：$\text{output} = F(x) + W_s x$。

- **瓶颈块**（用于 ResNet-50 及更深的网络）使用三个卷积：1x1 减少通道、3x3 进行空间处理、1x1 扩展回通道。这比两个 3x3 卷积更便宜，并允许更深的网络。

- **DenseNet**（Huang 等，2017）将跳跃连接思想进一步推进：在密集块内每一层都连接到后续的每一层。第 $l$ 层接收前面所有层的 feature map 作为输入：$x_l = H_l([x_0, x_1, \ldots, x_{l-1}])$，其中 $[\cdot]$ 表示沿通道维度拼接。这鼓励特征复用、增强梯度流动，并减少总参数量。

![DenseNet 密集块：每一层通过拼接接收来自前面所有层的 feature map，形成密集连接以最大化特征复用](../images/densenet_block.svg)

- **高效架构**面向移动设备和边缘硬件的部署，那里算力、内存和能量受限。

- **MobileNet**（Howard 等，2017）用**深度可分离卷积**（depthwise separable convolution）替代标准卷积，将操作分解为两步：
    1. **逐通道卷积**（depthwise convolution）：对每个输入通道应用一个单独的 $k \times k$ 滤波器（无跨通道交互）
    2. **逐点卷积**（pointwise convolution）：应用 1x1 卷积来跨通道组合信息

- 一个具有 $C_{\text{in}}$ 个输入通道和 $C_{\text{out}}$ 个输出通道的标准 $k \times k$ 卷积在每个空间位置花费 $k^2 \cdot C_{\text{in}} \cdot C_{\text{out}}$ 次乘法。深度可分离卷积花费 $k^2 \cdot C_{\text{in}} + C_{\text{in}} \cdot C_{\text{out}}$，减少了大约 $k^2$ 倍。对于 3x3 滤波器，大约便宜 9 倍。

![深度可分离卷积：逐通道步骤对每个通道应用一个 k×k 滤波器，然后逐点 1×1 卷积混合通道——相同输出形状，操作数约为 1/9](../images/depthwise_separable_conv.svg)

- **MobileNet-V2** 引入了**倒残差块**：用 1x1 卷积扩展通道，在扩展后的空间中应用深度卷积，再用 1x1 卷积投影回去。跳跃连接放在窄的（瓶颈）层上，反转了 ResNet 的模式。扩展比通常为 6。

- **EfficientNet**（Tan 和 Le，2019）引入了**复合缩放**：不再独立地只缩放深度、只缩放宽度或只缩放分辨率，而是以固定比例同时缩放三个维度。给定缩放系数 $\phi$：

$$\text{depth}: d = \alpha^\phi, \quad \text{width}: w = \beta^\phi, \quad \text{resolution}: r = \gamma^\phi$$

- 受限于 $\alpha \cdot \beta^2 \cdot \gamma^2 \approx 2$（使得总计算量在 $\phi$ 每增加 1 时大约翻倍）。网格搜索找到 $\alpha = 1.2$，$\beta = 1.1$，$\gamma = 1.15$ 作为基线比例。EfficientNet-B0 到 B7 逐步放大，以远少于以往模型的参数和 FLOPs 达到最先进的精度。

![EfficientNet 复合缩放：单独缩放宽度、深度或分辨率 vs 用单一系数 φ 同时缩放三者](../images/efficientnet_scaling.svg)

- **ShuffleNet** 通过使用**分组卷积**后接**通道重排**来降低 1x1 卷积的成本（在 MobileNet 风格架构中 1x1 卷积占主导）。分组卷积将通道分成若干组并在每组内独立卷积，但这阻碍了跨组信息流动。重排操作在组之间重新排列通道，以可忽略的代价恢复信息混合。

- **迁移学习**是将一个任务上训练好的模型适配到另一个任务上的实践。在计算机视觉中，这几乎总是指从一个在 ImageNet（140 万张图像，1000 个类）上预训练的模型出发，适配到特定领域的数据集（医学图像、卫星图像、制造缺陷）。

- **特征提取**：冻结所有卷积层，移除最后的分类头，只在其上训练一个新的头。被冻结的层充当通用特征提取器。当目标领域与 ImageNet 相似且目标数据集较小时效果很好。

- **微调（fine-tuning）**：解冻部分或全部卷积层，以小的学习率训练。预训练权重作为起点而非固定特征。微调通常先只解冻后面的层（捕获高层、任务特定特征），也可选择解冻更早的层。

- 迁移学习之所以有效，是因为 CNN 的早期层学习到跨任务通用的特征（边缘、纹理、颜色），而后面的层学习任务特定的特征。一个被训练来分类动物的网络仍然有用于分类建筑物的边缘检测器。

- **可视化 CNN** 揭示了网络学到了什么，有助于调试意外行为。

- **激活图**（feature map）显示每个滤波器对给定输入图像的输出。早期层激活看起来像边缘图；更深层的激活越来越抽象、空间上更粗糙。

- **Grad-CAM**（Gradient-weighted Class Activation Mapping，Selvaraju 等，2017）突出显示对模型预测最重要的输入图像区域。其工作方式是：
    1. 计算目标类别分数相对于最后一个卷积层 feature map 的梯度（使用第 03 章的链式法则）
    2. 对这些梯度做全局平均池化，得到每通道的重要性权重
    3. 计算feature map的加权和并应用 ReLU

$$L_{\text{Grad-CAM}} = \text{ReLU}\!\left(\sum_k \alpha_k A^k\right), \quad \alpha_k = \frac{1}{Z} \sum_i \sum_j \frac{\partial y^c}{\partial A^k_{ij}}$$

- 其中 $A^k$ 是第 $k$ 个 feature map，$\alpha_k$ 是通道 $k$ 的重要性权重，$y^c$ 是类别 $c$ 的分数。结果是一个粗略的热图，显示哪些区域驱动了分类。应用 ReLU 是因为我们关心对该类别有正向影响的特征。

![Grad-CAM：狗的输入图像、最后一个卷积层的 feature map、梯度加权组合以及叠加在原图上的热图，突出显示狗的脸](../images/grad_cam.svg)

- **特征反演**通过优化一张随机图像以匹配目标特征表示，从特征表示重建输入图像（对像素值进行梯度下降）。这揭示了网络在每层保留了什么信息。浅层重建出接近完美的图像；深层产生可辨认但扭曲的图像，表明精细的空间细节丢失了，而语义内容得以保留。

- **Deep Dream** 和**神经风格迁移**是特征可视化的创造性应用。Deep Dream 最大化所选层神经元的激活，产生超现实的、图案被放大的图像。神经风格迁移优化目标图像，使其匹配一张图像的内容特征（来自深层）和另一张图像的风格特征（滤波器激活的 Gram 矩阵，捕获纹理统计量）。

1. 在 JAX 中从零开始实现一个简单的 CNN，包含两个卷积层、最大池化和一个分类头。在合成的 2D 模式分类任务上训练它。
```python
import jax
import jax.numpy as jnp
import jax.lax as lax
import matplotlib.pyplot as plt

def conv2d(x, kernel, stride=1):
    """Simple 2D convolution for single input, single filter."""
    return lax.conv(x[None, None], kernel[None, None], (stride, stride), 'SAME')[0, 0]

def max_pool(x, size=2):
    """2x2 max pooling."""
    H, W = x.shape
    x = x[:H//size*size, :W//size*size]
    return x.reshape(H//size, size, W//size, size).max(axis=(1, 3))

def init_cnn(key):
    k1, k2, k3 = jax.random.split(key, 3)
    return {
        'conv1': jax.random.normal(k1, (5, 5)) * 0.3,
        'conv2': jax.random.normal(k2, (3, 3)) * 0.3,
        'fc_w': jax.random.normal(k3, (64, 1)) * 0.1,
        'fc_b': jnp.zeros(1),
    }

def forward_cnn(params, img):
    # Conv1 -> ReLU -> Pool
    h = jnp.maximum(0, conv2d(img, params['conv1']))
    h = max_pool(h)
    # Conv2 -> ReLU -> Pool
    h = jnp.maximum(0, conv2d(h, params['conv2']))
    h = max_pool(h)
    # Flatten and classify
    flat = h.ravel()
    # Pad or truncate to fixed size
    flat = jnp.pad(flat, (0, max(0, 64 - len(flat))))[:64]
    logit = (flat @ params['fc_w'] + params['fc_b']).squeeze()
    return jax.nn.sigmoid(logit)

# Generate synthetic data: class 0 = low-freq pattern, class 1 = high-freq
def make_data(key, n=200):
    images, labels = [], []
    for i in range(n):
        k1, key = jax.random.split(key)
        x, y = jnp.meshgrid(jnp.linspace(0, 4*jnp.pi, 32), jnp.linspace(0, 4*jnp.pi, 32))
        if i < n // 2:
            img = jnp.sin(x) + jax.random.normal(k1, (32, 32)) * 0.1
            labels.append(0)
        else:
            img = jnp.sin(4 * x) * jnp.sin(4 * y) + jax.random.normal(k1, (32, 32)) * 0.1
            labels.append(1)
        images.append(img)
    return images, jnp.array(labels, dtype=jnp.float32)

key = jax.random.PRNGKey(42)
images, labels = make_data(key)
params = init_cnn(jax.random.PRNGKey(0))

def loss_fn(params, img, label):
    pred = forward_cnn(params, img)
    return -(label * jnp.log(pred + 1e-7) + (1 - label) * jnp.log(1 - pred + 1e-7))

grad_fn = jax.grad(loss_fn)
lr = 0.01

for epoch in range(5):
    total_loss = 0.0
    for img, label in zip(images, labels):
        grads = grad_fn(params, img, label)
        params = {k: params[k] - lr * grads[k] for k in params}
        total_loss += loss_fn(params, img, label)
    print(f"Epoch {epoch}: loss = {total_loss / len(images):.4f}")

# Test accuracy
preds = jnp.array([forward_cnn(params, img) > 0.5 for img in images])
acc = jnp.mean(preds == labels)
print(f"Accuracy: {acc:.2%}")
```

2. 可视化不同滤波器大小如何影响感受野。展示两个堆叠的 3x3 滤波器与一个 5x5 滤波器覆盖相同的感受野，但参数更少。
```python
import jax.numpy as jnp
import matplotlib.pyplot as plt

def compute_receptive_field(layers):
    """Compute receptive field size from a list of (kernel_size, stride) tuples."""
    rf = 1  # start with 1 pixel
    stride_product = 1
    for k, s in layers:
        rf += (k - 1) * stride_product
        stride_product *= s
    return rf

# Compare architectures
configs = {
    'Single 5x5': [(5, 1)],
    'Two 3x3':    [(3, 1), (3, 1)],
    'Three 3x3':  [(3, 1), (3, 1), (3, 1)],
    'Single 7x7': [(7, 1)],
    '3x3 stride 2 + 3x3': [(3, 2), (3, 1)],
}

print(f"{'Config':<25} {'RF':>4} {'Params (per channel)':>20}")
print('-' * 55)
for name, layers in configs.items():
    rf = compute_receptive_field(layers)
    # Parameters: sum of k^2 for each layer (per input-output channel pair)
    params = sum(k * k for k, s in layers)
    print(f"{name:<25} {rf:>4} {params:>20}")

# Visualise receptive fields
fig, axes = plt.subplots(1, 3, figsize=(14, 4))
for ax, (name, rf_size) in zip(axes, [('5x5 filter', 5), ('Two 3x3 filters', 5), ('Three 3x3 filters', 7)]):
    grid = jnp.zeros((9, 9))
    c = 4  # centre
    half = rf_size // 2
    grid = grid.at[c-half:c+half+1, c-half:c+half+1].set(1.0)
    ax.imshow(grid, cmap='Blues', vmin=0, vmax=1)
    ax.set_title(f'{name}\nRF = {rf_size}x{rf_size}')
    ax.set_xticks(range(9)); ax.set_yticks(range(9))
    ax.grid(True, alpha=0.3)
plt.suptitle('Receptive Field Comparison')
plt.tight_layout(); plt.show()
```

3. 从零开始实现 Grad-CAM。给定一个预构建的简单 CNN，计算特定类别的梯度加权激活图，并可视化为热图。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

def simple_cnn(params, img):
    """Simple CNN that returns both the prediction and last conv activations."""
    # Conv layer (our "last conv layer" for Grad-CAM)
    H, W = img.shape
    k = params['conv'].shape[0]
    pad = k // 2
    img_pad = jnp.pad(img, pad, mode='edge')
    activation_map = jnp.zeros((H, W))
    for i in range(H):
        for j in range(W):
            activation_map = activation_map.at[i, j].set(
                jnp.sum(img_pad[i:i+k, j:j+k] * params['conv'])
            )
    activation_map = jnp.maximum(0, activation_map)  # ReLU

    # Global average pool -> dense -> output
    pooled = activation_map.mean()
    logit = pooled * params['w'] + params['b']
    return jax.nn.sigmoid(logit), activation_map

# Create test image: bright region on the left (class indicator)
img = jnp.zeros((32, 32))
img = img.at[8:24, 4:16].set(1.0)
img = img.at[5:10, 20:28].set(0.3)

key = jax.random.PRNGKey(42)
params = {
    'conv': jax.random.normal(key, (5, 5)) * 0.3,
    'w': jnp.array(2.0),
    'b': jnp.array(-0.5),
}

# Compute Grad-CAM
def class_score(params, img):
    pred, _ = simple_cnn(params, img)
    return pred

# Get activation map and gradients
pred, act_map = simple_cnn(params, img)
grad_fn = jax.grad(lambda img: simple_cnn(params, img)[0])
img_grad = grad_fn(img)

# Weight = global average of gradients (simplified 1-channel Grad-CAM)
alpha = img_grad.mean()
grad_cam = jnp.maximum(0, alpha * act_map)  # ReLU
grad_cam = (grad_cam - grad_cam.min()) / (grad_cam.max() - grad_cam.min() + 1e-8)

fig, axes = plt.subplots(1, 3, figsize=(14, 4))
axes[0].imshow(img, cmap='gray'); axes[0].set_title('Input Image'); axes[0].axis('off')
axes[1].imshow(act_map, cmap='viridis'); axes[1].set_title('Activation Map'); axes[1].axis('off')
axes[2].imshow(img, cmap='gray', alpha=0.6)
axes[2].imshow(grad_cam, cmap='jet', alpha=0.4)
axes[2].set_title(f'Grad-CAM (pred={pred:.2f})'); axes[2].axis('off')
plt.tight_layout(); plt.show()
```

4. 比较深度可分离卷积与标准卷积。计算两者的参数和 FLOPs，展示它们以远少的计算量产生相似的输出。
```python
import jax
import jax.numpy as jnp

def standard_conv(x, kernel):
    """Standard convolution: (H, W, C_in) * (k, k, C_in, C_out) -> (H, W, C_out)."""
    H, W, C_in = x.shape
    k, _, _, C_out = kernel.shape
    pad = k // 2
    x_pad = jnp.pad(x, ((pad, pad), (pad, pad), (0, 0)), mode='constant')
    out = jnp.zeros((H, W, C_out))
    for i in range(H):
        for j in range(W):
            patch = x_pad[i:i+k, j:j+k, :]  # (k, k, C_in)
            for c in range(C_out):
                out = out.at[i, j, c].set(jnp.sum(patch * kernel[:, :, :, c]))
    return out

def depthwise_separable_conv(x, dw_kernel, pw_kernel):
    """Depthwise separable: depthwise (k,k,C_in) then pointwise (C_in, C_out)."""
    H, W, C_in = x.shape
    k = dw_kernel.shape[0]
    pad = k // 2
    x_pad = jnp.pad(x, ((pad, pad), (pad, pad), (0, 0)), mode='constant')

    # Depthwise: one filter per channel
    dw_out = jnp.zeros((H, W, C_in))
    for i in range(H):
        for j in range(W):
            for c in range(C_in):
                patch = x_pad[i:i+k, j:j+k, c]
                dw_out = dw_out.at[i, j, c].set(jnp.sum(patch * dw_kernel[:, :, c]))

    # Pointwise: 1x1 conv across channels
    out = dw_out @ pw_kernel
    return out

# Setup
H, W, C_in, C_out, k = 8, 8, 16, 32, 3
key = jax.random.PRNGKey(42)
k1, k2, k3, k4 = jax.random.split(key, 4)

x = jax.random.normal(k1, (H, W, C_in))
std_kernel = jax.random.normal(k2, (k, k, C_in, C_out)) * 0.1
dw_kernel = jax.random.normal(k3, (k, k, C_in)) * 0.1
pw_kernel = jax.random.normal(k4, (C_in, C_out)) * 0.1

# Compare
std_params = k * k * C_in * C_out
dw_params = k * k * C_in + C_in * C_out

std_flops = H * W * k * k * C_in * C_out
dw_flops = H * W * (k * k * C_in + C_in * C_out)

print(f"Standard conv:            {std_params:>8,} params,  {std_flops:>10,} FLOPs")
print(f"Depthwise separable conv: {dw_params:>8,} params,  {dw_flops:>10,} FLOPs")
print(f"Parameter reduction:      {std_params / dw_params:.1f}x")
print(f"FLOP reduction:           {std_flops / dw_flops:.1f}x")

std_out = standard_conv(x, std_kernel)
ds_out = depthwise_separable_conv(x, dw_kernel, pw_kernel)
print(f"\nStandard output shape:    {std_out.shape}")
print(f"Depthwise sep output shape: {ds_out.shape}")
```
