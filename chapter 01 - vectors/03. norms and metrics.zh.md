# 度量与范数

*norm 衡量一个向量的大小；metric 衡量两个向量之间的距离。本文件涵盖 L1、L2 与 L-infinity norm，Euclidean 与 cosine distance，以及为什么选择正确的距离函数对 kNN、聚类和 ML 中的检索至关重要。*

- 我们已经知道向量有 magnitude 和 direction。但究竟该如何度量一个向量“有多大”，或两个向量“相距多远”？这就是 **norm** 和 **metric** 出场的地方。

- 对于 scalar，我们知道 10 > 5，因为它们的值就量化了它们，但如何量化一个向量？用它的 **norm**，它衡量单个向量的大小。

- 最熟悉的 norm 是 **Euclidean norm**（L2），也就是我们已经熟悉的 magnitude 公式：

$$\|\mathbf{v}\|_2 = \sqrt{v_1^2 + v_2^2 + \cdots + v_n^2}$$

- 但衡量大小还有别的方式。想象你在一座街道呈网格状的城市里。你无法斜穿建筑走对角线，所以旅程的“长度”是沿每条街所走的街区总和。这就是 **Manhattan norm**（L1）：

$$\|\mathbf{v}\|_1 = |v_1| + |v_2| + \cdots + |v_n|$$

- 或者你只关心最大的那一个分量，忽略其他。这就是 **Max Norm**（L-infinity）：

$$\|\mathbf{v}\|_\infty = \max(|v_1|, |v_2|, \ldots, |v_n|)$$

- 这三种都是 **广义 Lp norm** 的特例：

$$\|\mathbf{v}\|_p = (|v_1|^p + |v_2|^p + \cdots + |v_n|^p)^{1/p}$$

- 令 $p = 2$ 得 Euclidean，$p = 1$ 得 Manhattan，当 $p \to \infty$ 时得到 Max norm。随着 $p$ 增大，最大分量的贡献越来越大，直至最终只有它起作用。

- 每个 norm 都必须遵循三条规则：

    - **非负性**：$\|\mathbf{v}\| \geq 0$，且仅当 $\mathbf{v} = \mathbf{0}$ 时 $\|\mathbf{v}\| = 0$。大小永不为负，只有零向量大小为零。

    - **缩放性**：$\|c\mathbf{v}\| = |c| \cdot \|\mathbf{v}\|$。向量加倍，大小也加倍。

    - **三角不等式**：$\|\mathbf{u} + \mathbf{v}\| \leq \|\mathbf{u}\| + \|\mathbf{v}\|$。捷径永远不会比绕远路更长。

- 而 **metric** 衡量的是两个向量 *之间* 的距离。可以把它想成在问：“这两个点相距多远？”

- 获得一个 metric 的最简单办法就是对差值应用一个 norm：$d(\mathbf{u}, \mathbf{v}) = \|\mathbf{u} - \mathbf{v}\|$。把两个向量相减，再衡量剩余部分的大小。

- 使用 Euclidean norm，就得到我们熟悉的 **Euclidean distance**：

$$d(\mathbf{u}, \mathbf{v}) = \sqrt{(u_1 - v_1)^2 + (u_2 - v_2)^2 + \cdots + (u_n - v_n)^2}$$

- 使用 Manhattan norm 得到 **Manhattan distance**，即各轴差异的总和，就像在数两个地点之间的城市街区数。

- 每个 metric 都必须遵循四条规则：

    - **非负性**：$d(\mathbf{u}, \mathbf{v}) \geq 0$。距离永不为负。

    - **同一性**：当且仅当 $\mathbf{u} = \mathbf{v}$ 时 $d(\mathbf{u}, \mathbf{v}) = 0$。距离为零意味着同一个点。

    - **对称性**：$d(\mathbf{u}, \mathbf{v}) = d(\mathbf{v}, \mathbf{u})$。从 A 到 B 的距离与从 B 到 A 相同。

    - **三角不等式**：$d(\mathbf{u}, \mathbf{w}) \leq d(\mathbf{u}, \mathbf{v}) + d(\mathbf{v}, \mathbf{w})$。直接走永远不会比绕道更长。

- 那么两者之间是什么关系？norm 衡量一个向量，metric 衡量两个向量之间的差距。每个 norm 都自然派生出一个 metric（通过衡量差值），但并非每个 metric 都来自一个 norm。

- 例如，**Hamming distance** 统计两个向量取值不同的位置数目。它是一个合法的 metric，但不来自任何 norm。

- 在 ML 中，选择正确的 norm 或 metric 至关重要。

- L2 distance 在求和前对每个差值取平方，因此一个大的差异会主导结果。

- L1 distance 对绝对差值求和，平等对待每个差值。相比 L2，单个大的差异影响较小。

## 编程任务（使用 CoLab 或 notebook）

1. 计算同一向量的 L1 与 L2 norm。试着改变数值，留意哪个 norm 对大分量更敏感，哪个对许多小分量更敏感。再尝试对 p 取递增值（例如 1、2、5、10、50、100）计算 Lp norm，观察它如何向 L-infinity 值收敛。
```python
import jax.numpy as jnp

v = jnp.array([3.0, -4.0, 1.0])

l1 = jnp.sum(jnp.abs(v))
l2 = jnp.sqrt(jnp.sum(v ** 2))

print(f"L1: {l1}, L2: {l2:.2f}")
```

2. 计算两个向量之间的 Euclidean 和 Manhattan distance。试着把两个向量拉近或拉远，观察每种距离的不同响应。
```python
import jax.numpy as jnp

u = jnp.array([1.0, 2.0, 3.0])
v = jnp.array([4.0, 0.0, 1.0])

euclidean = jnp.sqrt(jnp.sum((u - v) ** 2))
manhattan = jnp.sum(jnp.abs(u - v))

print(f"Euclidean: {euclidean:.2f}, Manhattan: {manhattan}")
```
