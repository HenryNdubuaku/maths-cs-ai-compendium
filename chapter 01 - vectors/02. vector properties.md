# Свойства векторов

*Свойства векторов описывают геометрические и алгебраические характеристики, определяющие поведение векторов. В этом файле рассматриваются величина, направление, единичные векторы, равенство, параллельность, ортогональность и линейная независимость — фундаментальные элементы любого пространства признаков в машинном обучении.*

- **Величина** (или длина) вектора показывает, *как далеко* он простирается. Представьте её как длину стрелки. Для вектора $\mathbf{a} = (a_1, a_2, a_3)$ его величина равна:

$$\|\mathbf{a}\| = \sqrt{a_1^2 + a_2^2 + a_3^2}$$

- Это просто теорема Пифагора, расширенная на пространства большей размерности для измерения расстояния по прямой от начала координат до точки.

- **Направление** вектора показывает, *куда* он указывает; просто представьте прямую линию от начала координат до точки с данными координатами.

- Когда начало координат не задано явно, мы часто подразумеваем $(0,0,...0)$, центральную точку, по крайней мере для целей визуализации.

- Положение не имеет значения, речь всегда идет о смещении: вектор $(3, 2)$, проведенный из начала координат, и такой же вектор $(3, 2)$, проведенный из другой точки, всё равно равны.

![Равенство векторов: один и тот же вектор (3,2), проведенный из двух разных начальных точек](../images/vector_equality.svg)

- Два вектора могут иметь одинаковую длину, но указывать в совершенно разных направлениях, или указывать в одном направлении, но отличаться по длине.

![Одинаковое направление, разные величины (v и 2v) против одинаковой величины, разные направления](../images/magnitude_direction.svg)

- Два вектора **равны** тогда и только тогда, когда все их соответствующие компоненты совпадают; одинаковая длина, одинаковое направление, одна и та же стрелка.

$$\mathbf{a} = \mathbf{b} \iff a_i = b_i \text{ for all } i$$

- Два вектора **параллельны**, если один является скалярным кратным другого. Они лежат на одной прямой, указывая либо в одном направлении, либо в прямо противоположных.

$$\mathbf{a} \parallel \mathbf{b} \iff \mathbf{a} = k\mathbf{b} \text{ for some scalar } k \neq 0$$

![Параллельные векторы: a и b указывают в одну сторону, a и -b указывают в противоположные](../images/parallel_vectors.svg)

- Если $k > 0$, они указывают в одну сторону. Если $k < 0$, они указывают в противоположных направлениях. В любом случае они лежат на одной прямой, проходящей через начало координат.

- Интуитивно понятно, что параллельные векторы не несут «новой» информации о направлении. Один вектор — это просто растянутая или перевернутая версия другого.

- Два вектора **ортогональны** (перпендикулярны), если они указывают в совершенно независимых направлениях. Движение вдоль одного вектора дает нулевое продвижение вдоль другого.

![Ортогональные векторы: u и v пересекаются под прямым углом](../images/orthogonal_vectors.svg)

- Представьте, что вы идете на север, а затем на восток — это ортогональные направления; никакое движение на север никогда не переместит вас на восток. Мы будем очень часто сталкиваться с ортогональностью.

- Ортогональность имеет решающее значение для машинного обучения: признаки, которые ортогональны, несут полностью независимую информацию, что идеально подходит для представления данных.

- В более общем случае любые два вектора имеют **угол** $\theta$ между ними, варьирующийся от $0^\circ$ до $180^\circ$.

- Этот угол отражает всю взаимосвязь между двумя направлениями: $0^\circ$ означает параллельность (одно направление), $180^\circ$ означает параллельность (противоположное направление), а $90^\circ$ означает ортогональность. Всё, что находится между ними, является комбинацией этих состояний.

- Большинство векторных отношений в машинном обучении находятся где-то в этом спектре. Позже мы рассмотрим точные инструменты (скалярное произведение, косинусное сходство) для вычисления этого угла.

- Набор векторов является **линейно зависимым**, если хотя бы один из них может быть получен из остальных путем масштабирования и сложения. Он не привносит в набор никакой новой информации.

- Например, если $\mathbf{c} = 2\mathbf{a} + 3\mathbf{b}$, то $\mathbf{c}$ избыточен: у вас уже есть всё, что предлагает $\mathbf{c}$, через $\mathbf{a}$ и $\mathbf{b}$.

- Параллельные векторы всегда линейно зависимы, так как один является просто масштабированной копией другого. Любой набор, содержащий нулевой вектор, также является линейно зависимым.

- Векторы **линейно независимы**, если ни один из них не может быть получен из остальных. Каждый из них вносит принципиально новое направление. Ортогональные векторы всегда линейно независимы.

- Немного интуиции: если вы хотите изучить разных людей и представить их в виде вектора, линейно зависимые векторы (люди) сместят наблюдение в пользу перепредставленной точки данных, что является важным фактором при проектировании датасета для обучения ИИ.

- В 2D два линейно независимых вектора могут достичь любой точки на плоскости. В 3D вам нужно три. Эта идея о том, «сколько независимых векторов вам нужно», напрямую связана с размерностью.

- Вектор является **разреженным**, когда большинство его компонентов равны нулю. Противоположный случай, когда большинство компонентов ненулевые, называется **плотным**.

$$\mathbf{s} = [0, 0, 3, 0, 0, 0, 1, 0, 0, 0]$$

- Разреженность важна, так как она влияет как на хранение, так и на вычисления. Разреженные векторы могут храниться и обрабатываться гораздо эффективнее за счет отслеживания только ненулевых элементов.

- **Единичный вектор** — это вектор с величиной, равной ровно 1. Он чисто представляет направление без информации о длине. Вы можете превратить любой вектор в единичный, разделив его на его величину:

$$\hat{\mathbf{a}} = \frac{\mathbf{a}}{\|\mathbf{a}\|}$$

- Этот процесс называется **нормализацией**. Он отбрасывает информацию о том, «как далеко», и оставляет только «в какую сторону»; это важный фактор в машинном обучении.

- Стандартные единичные векторы указывают вдоль каждой оси: $\hat{\mathbf{i}} = (1, 0, 0)$, $\hat{\mathbf{j}} = (0, 1, 0)$, $\hat{\mathbf{k}} = (0, 0, 1)$. Любой вектор может быть записан как комбинация этих векторов, например $(3, 2, 4) = 3\hat{\mathbf{i}} + 2\hat{\mathbf{j}} + 4\hat{\mathbf{k}}$.

## Задачи по программированию (используйте CoLab или ноутбук)

1. Вычислите величину вектора и убедитесь, что она соответствует теореме Пифагора, затем модифицируйте код для вычисления единичного вектора.
```python
import jax.numpy as jnp

a = jnp.array([3.0, 4.0])

magnitude = jnp.sqrt(jnp.sum(a ** 2))
print(f"Magnitude of a: {magnitude}") 
```

2. Проверьте, являются ли два вектора параллельными, протестировав, является ли один из них скалярным кратным другого.
```python
import jax.numpy as jnp

a = jnp.array([2, 4, 6])
b = jnp.array([1, 2, 3])

ratios = a / b
print(f"Ratios: {ratios}")
print(f"Parallel: {jnp.allclose(ratios, ratios[0])}")
```
