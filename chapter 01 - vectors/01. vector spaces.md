# Векторные пространства

*Векторные пространства образуют математическую «игровую площадку», на которой существует машинное обучение. В этом файле рассматриваются сложение векторов, умножение на скаляр, аксиомы замкнутости, подпространства и то, почему почти всё в ИИ представляется в виде векторов.*

- Представьте векторное пространство как своего рода игровую площадку, где живут математические объекты, и каждый такой объект называется **вектором**.

- Векторное пространство формально определяется как совокупность векторов, которые можно складывать и масштабировать, не выходя за пределы этого пространства.

- Полезный контрпример: целые числа $\mathbb{Z}$ **не** являются векторным пространством над полем вещественных чисел, так как при умножении $3$ на $0.5$ получается $1.5$, что находится вне данного множества. Условие «не выходя за пределы пространства» в определении играет ключевую роль.

- Для геометрической интуиции в машинном обучении (ML) мы всегда будем рассматривать векторы как точки в евклидовом пространстве, представленные своими координатами.

- Вектор $\mathbf{a}$ (математически обозначаемый строчными буквами, выделенными жирным шрифтом) имеет $n$ координат, каждая из которых представляет положение вдоль соответствующей оси.

$$\mathbf{a} = [a_1, a_2, a_3]$$

![Вектор a = (3, 2, 4), построенный в трехмерном пространстве с осями x, y, z](../images/vector_3d.svg)

- Векторы в векторном пространстве подчиняются очень специфическому, незыблемому набору правил:

    - **Сложение векторов (комбинирование)**:
    Вы можете взять любые два вектора и объединить их, чтобы создать новый.
    Представьте векторы как инструкции для движения.
    Если вектор A означает «пройти 3 шага вперед», а вектор B — «пройти 2 шага вправо»,
    то их сложение (A + B) создает новую единую инструкцию: «пройти 3 шага вперед и 2 шага вправо».

    - **Умножение на скаляр (масштабирование)**:
    Вы можете взять любой вектор и масштабировать его с помощью обычного числа («скаляра»).
    Вы можете растянуть его, сжать или развернуть в обратную сторону.
    Если вектор A — это «пройти 3 шага вперед», то масштабирование на 2 превращает его в «пройти 6 шагов вперед».
    Масштабирование на -1 полностью разворачивает его в «пройти 3 шага назад».

- **Размерность** векторного пространства — это количество независимых направлений, которые оно содержит. $\mathbb{R}^2$ является двумерным (требует 2 координаты), в то время как вектор $\mathbf{a}$ выше существует в $\mathbb{R}^3$.

- Мы можем, например, представить любой объект, скажем, человека, как вектор, где $h_1$ = рост в см, $h_2$ = вес в кг, $h_3$ = возраст.

$$\mathbf{h} = [185, 75, 30]$$

- Теперь мы создали векторное пространство, в котором вектор представляет человека.

- Мы можем представить нескольких людей и увидеть, насколько они близки или далеки друг от друга!

![Три человека в виде векторов: Элис и Кэрол близки, Элис и Боб находятся далеко друг от друга](../images/human_vectors.svg)

- Мы можем добавить больше признаков, создав богатое представление человека, которое в ML часто называют вектором признаков.

- Чем больше уникальных и значимых признаков вы имеете, тем более описательным является вектор признаков — это важный фактор, который следует помнить.

- За пределами 3 измерений векторы становятся очень сложными для визуального анализа, что вдохновило создание раздела математики под названием **линейная алгебра**.

- Итак, **линейная алгебра** — это изучение векторов, векторных пространств и отображений между векторами.

- Мы представляем почти всё в ИИ/ML как векторы, что делает линейную алгебру фундаментом этой области.

- Сложение векторов можно выполнить визуально, приставив начало одного вектора к концу другого и проведя результирующий вектор от начала координат до конечной точки.

![Сложение векторов: a (красный) плюс b (синий) дает результат a + b (зеленый пунктир)](../images/vector_addition.svg)

- Для двух векторов $\mathbf{a} = (a_1, a_2)$ и $\mathbf{b} = (b_1, b_2)$: $\mathbf{a} + \mathbf{b} = (a_1 + b_1, a_2 + b_2)$

- Векторы также можно вычитать, при этом применяются все правила сложения.

- Умножение вектора на скаляр масштабирует вектор на этот коэффициент в том же направлении.

![Умножение на скаляр: v (красный), 2v (синий, удвоенный), -v (фиолетовый, развернутый)](../images/scalar_multiplication.svg)

- Для скаляра $c$ и вектора $\mathbf{v} = (v_1, v_2)$: $c\mathbf{v} = (cv_1, cv_2)$

- **Замкнутость относительно сложения**: Если вы складываете любые два вектора из векторного пространства, результат также является вектором внутри того же пространства: Если $\mathbf{u} \in V$ и $\mathbf{v} \in V$, то $\mathbf{u} + \mathbf{v} \in V$

- **Замкнутость относительно умножения на скаляр**: Если вы умножаете любой вектор из векторного пространства на скаляр, результат является вектором внутри того же пространства: Если $\mathbf{v} \in V$ и $c \in F$, то $c\mathbf{v} \in V$

- **Коммутативность сложения**: Для любых двух векторов $\mathbf{u}$ и $\mathbf{v}$: $\mathbf{u} + \mathbf{v} = \mathbf{v} + \mathbf{u}$

![Правило параллелограмма: оба пути (u затем v, или v затем u) приводят в одну и ту же точку](../images/commutativity.svg)

- Оба пути через параллелограмм приводят в одну и ту же точку.

- **(Нулевой вектор)**: Существует вектор $\mathbf{0}$ такой, что для любого вектора $\mathbf{v}$: $\mathbf{v} + \mathbf{0} = \mathbf{v}$

![Нулевой вектор: v + 0 = v](../images/zero_vector.svg)

- **Аддитивно-обратный элемент**: Для каждого вектора $\mathbf{v}$ существует вектор $-\mathbf{v}$ такой, что: $\mathbf{v} + (-\mathbf{v}) = \mathbf{0}$

![Аддитивно-обратный элемент: v (красный) и -v (синий) при сложении дают ноль](../images/additive_inverse.svg)

- **Дистрибутивность 1**: Для любого скаляра $c$ и векторов $\mathbf{u}$, $\mathbf{v}$: $c(\mathbf{u} + \mathbf{v}) = c\mathbf{u} + c\mathbf{v}$

![Дистрибутивность: масштабирование суммы (золотой) равно сумме масштабированных векторов](../images/distributivity.svg)

- Масштабирование суммы (золотой) дает тот же результат, что и суммирование масштабированных векторов.

- **Дистрибутивность 2**: Для любых скаляров $c$, $d$ и вектора $\mathbf{v}$: $(c + d)\mathbf{v} = c\mathbf{v} + d\mathbf{v}$

- **Ассоциативность**: Для любых скаляров $c$, $d$ и вектора $\mathbf{v}$: $(cd)\mathbf{v} = c(d\mathbf{v})$

- **Нейтральный элемент**: Для любого вектора $\mathbf{v}$: $1\mathbf{v} = \mathbf{v}$, где $1$ — мультипликативная единица в поле скаляров.

- Некоторые примеры векторных пространств:

    - **$\mathbb{R}^n$ (n-мерное пространство)**: все вещественные числа в n-мерном пространстве, пример вектора [1,4,3000...n-й элемент], сложите две точки или масштабируйте одну, и вы все равно окажетесь где-то в этой плоскости.

    - **Изображения в оттенках серого**: изображение $28 \times 28$ — это просто 784 интенсивности пикселей, то есть вектор в $\mathbb{R}^{784}$. Сложение двух изображений (смешивание) или масштабирование одного (изменение яркости) дает другое изображение того же размера.

- **Аудиосигналы**: 1-секундный фрагмент с частотой дискретизации 44,1 кГц — это вектор из 44 100 элементов. Смешивание двух фрагментов — это просто сложение векторов.

- **Многочлены**: сложение двух многочленов или умножение одного из них на число дает другой многочлен, поэтому они также образуют векторное пространство. Векторы не обязательно должны выглядеть как стрелки!

- **Подпространство** — это просто «игровая площадка» поменьше внутри более крупной. Представьте 3D-пространство как комнату. Плоский лист бумаги, проходящий через центр комнаты, является подпространством; то же самое можно сказать и об одной прямой проволоке, проходящей через центр.

- Ключевое требование заключается в том, что подпространство должно проходить через начало координат. Если сместить этот лист бумаги от центра, он перестает быть подпространством, так как нулевой вектор больше ему не принадлежит.

![Подпространства: прямая и плоскость, проходящие через начало координат внутри 3D-пространства](../images/subspaces.svg)

- Все те же правила векторного пространства (сложение, масштабирование, замкнутость) продолжают работать внутри подпространства. Вы можете складывать или масштабировать векторы внутри него и никогда не «выпадете» в объемлющее пространство.

- Прямая, проходящая через начало координат, — это 1-мерное подпространство, плоскость, проходящая через начало координат, — это 2-мерное подпространство, а всё пространство целиком является подпространством самого себя.

- В машинном обучении подпространства возникают естественным образом. Высокоразмерные данные часто обладают структурой, которая лежит в подпространстве меньшей размерности. Такие методы, как PCA, находят это подпространство, чтобы мы могли работать с данными более эффективно.

## Задачи по программированию (используйте CoLab или ноутбук)

1. Запустите код, чтобы проверить свойство дистрибутивности, а затем измените его и поэкспериментируйте, чтобы протестировать другие правила!
```python
import jax.numpy as jnp

u = jnp.array([1, 2])
v = jnp.array([3, 0])
c = 2

lhs = c * (u + v)
rhs = c*u + c*v

print(f"LHS: {lhs}")
print(f"RHS: {rhs}")
```

2. Запустите код для визуализации различных векторов, а затем измените значения координат, чтобы понять, как каждая ось влияет на положение.
```python
import jax.numpy as jnp
import matplotlib.pyplot as plt

# Try changing these vectors!
a = jnp.array([3, 2, 4])
b = jnp.array([1, 4, 2])
c = jnp.array([4, 1, 3])

fig = plt.figure()
ax = fig.add_subplot(111, projection="3d")

for vec, name, color in [(a, "a", "red"), (b, "b", "blue"), (c, "c", "green")]:
    ax.quiver(0, 0, 0, *vec, color=color, arrow_length_ratio=0.1, linewidth=2, label=name)

lim = int(jnp.abs(jnp.stack([a, b, c])).max()) + 1
ax.set_xlim([0, lim]); ax.set_ylim([0, lim]); ax.set_zlim([0, lim])
ax.set_xlabel("X"); ax.set_ylabel("Y"); ax.set_zlabel("Z")
ax.legend()
plt.show()
```

## Дополнительное задание (математические символы, которые стоит знать)

| Символ | Значение | Пример |
|---|---|---|
| ∈ | принадлежит множеству | x ∈ A: x находится в множестве A |
| ∉ | не принадлежит множеству | x ∉ A |
| ⊂ | является собственным подмножеством | A ⊂ B |
| ⊆ | является подмножеством или равно | A ⊆ B |
| ∪ | объединение | A ∪ B: элементы, принадлежащие A или B |
| ∩ | пересечение | A ∩ B: элементы, принадлежащие и A, и B |
| ∅ | пустое множество | A = ∅ |
| ℝ | вещественные числа | x ∈ ℝ |
| ℤ | целые числа | -2, -1, 0, 1, 2 |
| ℕ | натуральные числа | 1, 2, 3, ... |
| ℚ | рациональные числа | дроби, например 1/2 |
| ⇒ | влечет за собой | x > 2 ⇒ x > 1 |
| ⇔ | тогда и только тогда, когда | x = 2 ⇔ x² = 4, при дополнительных условиях |
| ∀ | для всех | ∀x ∈ ℝ |
| ∃ | существует | ∃x такое, что x² = 4 |
| ¬ | не | ¬P: не P |
| ∧ | и | P ∧ Q |
| ∨ | или | P ∨ Q |
| ∴ | следовательно | x = 2, ∴ x² = 4 |
