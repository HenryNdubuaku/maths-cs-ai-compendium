# Интегральное исчисление

*Интегральное исчисление позволяет суммировать величины на интервалах, преобразуя локальные скорости изменения обратно в итоговые значения. В этом файле рассматриваются определённые и неопределённые интегралы, основная теорема математического анализа, методы интегрирования, а также их применение к плотностям вероятности и математическим ожиданиям в машинном обучении.*

- Дифференцирование показывает скорость изменения в одной точке. Интегрирование работает в обратном направлении: оно суммирует множество крошечных частей для вычисления общего значения.

- Если производная отвечает на вопрос «как быстро?», то интеграл отвечает на вопрос «сколько?».

- Проще всего представить интегрирование как **площадь под кривой**. Если построить график функции $f(x)$ и заштриховать область между кривой и осью x от $x = a$ до $x = b$, то интеграл даст значение площади этой области с учётом знака.

![Интегрирование вычисляет площадь под кривой путём суммирования узких прямоугольников](../images/area_under_curve.svg)

- Почему «с учётом знака»? Области над осью x дают положительную площадь, области под ней — отрицательную. Это имеет физический смысл: если $f(x)$ представляет скорость, то интеграл даёт чистое перемещение (вперёд минус назад), а не пройденный путь.

- Чтобы вычислить эту площадь, представьте, что область разбита на $n$ узких вертикальных прямоугольников шириной $\Delta x$ каждый. Высота каждого прямоугольника равна значению функции в некоторой точке этого среза. Просуммируем их:

$$\text{Area} \approx \sum_{i=1}^{n} f(x_i^\ast) \, \Delta x$$

- По мере того как мы делаем прямоугольники всё тоньше ($n \to \infty$, $\Delta x \to 0$), сумма становится точной. Этот предельный процесс определяет **определённый интеграл**:

$$\int_a^b f(x)\, dx = \lim_{n \to \infty} \sum_{i=1}^{n} f(x_i^\ast) \, \Delta x$$

- Символ $\int$ — это вытянутая «S», означающая «sum» (сумма). Запись $dx$ напоминает нам, что мы суммируем бесконечно тонкие срезы вдоль оси x.

- **Неопределённый интеграл** (или **первообразная**) — это функция $F(x)$, производная которой равна $f(x)$. Мы записываем это так:

$$\int f(x)\, dx = F(x) + C$$

- $+ C$ — это **константа интегрирования**. Поскольку производная любой константы равна нулю, существует бесконечно много первообразных, отличающихся только на константу. Например, $\int 2x\, dx = x^2 + C$, так как производная $x^2 + 7$ или $x^2 - 3$ по-прежнему равна $2x$.

- **Основная теорема математического анализа** — это мост, соединяющий дифференцирование и интегрирование. Она состоит из двух частей:

- **Часть 1**: Если $F(x)$ является первообразной для $f(x)$, то определённый интеграл равен разности значений $F$ на концах интервала:

$$\int_a^b f(x)\, dx = F(b) - F(a)$$

- Это чрезвычайно практично. Вместо вычисления предела сумм (что сложно), мы находим первообразную и вычисляем её значение в двух точках (что обычно легко).

- **Часть 2**: Если определить $F(x) = \int_a^x f(t)\, dt$, то $F'(x) = f(x)$. Дифференцирование и интегрирование — это обратные операции, они взаимно отменяют друг друга.

- Например, вычислим $\int_1^3 x^2\, dx$: первообразная для $x^2$ равна $\frac{x^3}{3}$. Тогда $\int_1^3 x^2\, dx = \frac{27}{3} - \frac{1}{3} = \frac{26}{3} \approx 8.67$.

- Подобно тому как у дифференцирования есть правила, у интегрирования есть соответствующие правила, которые их обращают:

| Функция | Интеграл | Условие |
|---|---|---|
| $x^n$ | $\frac{x^{n+1}}{n+1} + C$ | $n \neq -1$ |
| $\frac{1}{x}$ | $\ln\|x\| + C$ | |
| $e^x$ | $e^x + C$ | |
| $a^x$ | $\frac{a^x}{\ln a} + C$ | |
| $\sin x$ | $-\cos x + C$ | |
| $\cos x$ | $\sin x + C$ | |
| $k$ (константа) | $kx + C$ | |

- **Правило суммы/разности** сохраняется: $\int [f(x) \pm g(x)]\, dx = \int f(x)\, dx \pm \int g(x)\, dx$. Константы можно выносить за знак интеграла: $\int k\, f(x)\, dx = k \int f(x)\, dx$.

- Когда функция слишком сложна для прямого интегрирования, существуют методы её упрощения.

- **Метод замены переменной (u-substitution)** — это обращение цепного правила. Если вы видите сложную функцию $f(g(x))$, умноженную на $g'(x)$, сделайте замену $u = g(x)$, тогда $du = g'(x)\, dx$, и интеграл упростится.

- Например: $\int 2x \cos(x^2)\, dx$. Пусть $u = x^2$, тогда $du = 2x\, dx$. Интеграл принимает вид $\int \cos(u)\, du = \sin(u) + C = \sin(x^2) + C$.

- **Интегрирование по частям** — это обращение правила произведения. Если подынтегральное выражение является произведением двух функций:

$$\int u\, dv = uv - \int v\, du$$

- Выбирайте $u$ и $dv$ стратегически так, чтобы оставшийся интеграл $\int v\, du$ был проще исходного. Популярная мнемоника для выбора $u$ — **LIATE**: логарифмические, обратные тригонометрические, алгебраические, тригонометрические, экспоненциальные (выбирайте $u$ из категории, стоящей раньше в списке).

- Например: $\int x\, e^x\, dx$. Пусть $u = x$ (алгебраическая) и $dv = e^x\, dx$. Тогда $du = dx$ и $v = e^x$. Получаем: $\int x\, e^x\, dx = x\, e^x - \int e^x\, dx = x\, e^x - e^x + C = e^x(x - 1) + C$.

- В машинном обучении интегрирование встречается в теории вероятностей (вычисление вероятностей путём интегрирования функций плотности), при расчёте математических ожиданий (взвешенные средние по непрерывным распределениям) и при вычислении площади под ROC-кривыми. Хотя на практике мы редко интегрируем вручную, понимание смысла интегрирования помогает интерпретировать эти величины.

## Задачи по программированию (используйте CoLab или ноутбук)

1. Численно аппроксимируйте $\int_0^1 x^2\, dx$, используя интегральную сумму Римана с возрастающим числом прямоугольников. Сравните с точным ответом $\frac{1}{3}$.
```python
import jax.numpy as jnp

for n in [10, 100, 1000, 10000]:
    x = jnp.linspace(0, 1, n, endpoint=False)
    dx = 1.0 / n
    area = jnp.sum(x**2 * dx)
    print(f"n={n:5d}  approx: {area:.6f}  exact: {1/3:.6f}")
```

2. Проверьте основную теорему математического анализа численно. Определите $F(x) = \int_0^x t^2\, dt = \frac{x^3}{3}$ и убедитесь, что её производная (вычисленная через `jax.grad`) равна $x^2$.
```python
import jax
import jax.numpy as jnp

F = lambda x: x**3 / 3
dF = jax.grad(F)

for x in [0.5, 1.0, 2.0, 3.0]:
    print(f"x={x:.1f}  F'(x)={dF(x):.4f}  x^2={x**2:.4f}")
```

3. Визуализируйте площадь под $f(x) = \sin(x)$ от $0$ до $\pi$. Используйте `plt.fill_between` для закрашивания области и вычислите её численно с помощью суммы Римана.
```python
import jax.numpy as jnp
import matplotlib.pyplot as plt

x = jnp.linspace(0, jnp.pi, 500)
y = jnp.sin(x)

plt.plot(x, y, color="purple", linewidth=2)
plt.fill_between(x, y, alpha=0.2, color="purple")
plt.title(f"Area = {jnp.sum(jnp.sin(x) * (jnp.pi / 500)):.4f}  (exact: 2.0)")
plt.show()
```
