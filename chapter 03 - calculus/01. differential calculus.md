# Дифференциальное исчисление

*Дифференциальное исчисление описывает мгновенные скорости изменения. В этом файле рассматриваются пределы, производные, правила дифференцирования, цепное правило (основа обратного распространения ошибки) и распространенные производные, используемые в машинном обучении.*

- В предыдущих главах мы узнали, как представлять данные в виде векторов и преобразовывать их с помощью матриц. Однако многие явления реального мира не являются статичными. Автомобиль ускоряется, цена акций колеблется, функция потерь нейронной сети меняется по мере обновления весов. **Математический анализ** (calculus) — это математика изменений.

- Математический анализ ставит два вопроса: как быстро что-то меняется прямо сейчас? (дифференциальное исчисление) и сколько накопилось за определенное время? (интегральное исчисление). Этот раздел посвящен вопросу «как быстро».

- Представьте, что вы ведете машину и смотрите на спидометр. Он показывает 60 км/ч. Это число — не средняя скорость всей поездки, а ваша скорость в данный конкретный момент. Дифференциальное исчисление дает нам инструменты для вычисления таких мгновенных скоростей изменения.

- Но сначала давайте вспомним уравнение прямой: $y = mx + b$.

- Это простейшая зависимость между двумя величинами.

    - $b$ — это **y-перехват**, точка, в которой прямая пересекает ось y (начальное значение при $x = 0$).
    - $m$ — это **наклон** (коэффициент наклона), скорость изменения: на каждую единицу увеличения $x$ значение $y$ изменяется на $m$.
- Если $m = 3$, прямая круто идет вверх; если $m = 0$, прямая горизонтальна; если $m = -2$, прямая идет вниз.

- Наклон вычисляется как $m = \frac{\Delta y}{\Delta x} = \frac{y_2 - y_1}{x_2 - x_1}$, отношение «насколько изменился $y$» к «насколько изменился $x$».

![Уравнение прямой: b — y-перехват, m — наклон (отношение вертикального изменения к горизонтальному)](../images/line_equation.svg)

- Зная $m$ и $b$, можно вычислить $y$ для любого $x$.

- Например, если $m = 2$ и $b = 3$, то при $x = 5$: $y = 2(5) + 3 = 13$.

- Эти два параметра полностью определяют прямую, и для предсказания любого выхода достаточно просто подставить значения.

- Для прямой линии наклон везде одинаков.

- Эта идея обобщается и за пределы прямых. Любая функция — это правило, которое отображает входы в выходы, и как только вы знаете ее формулу (ее параметры и форму), вы можете вычислить выход для любого входа и построить график результата.

- $y = x^2$ дает параболу, $y = \sin(x)$ дает волну, $y = e^x$ дает экспоненциальный рост. Каждая формула определяет конкретную кривую, и умение «читать» функцию как форму графика необходимо для всего, что будет дальше.

- Для прямой линии наклон везде одинаков. Но большинство интересных функций являются кривыми, поэтому наклон меняется от точки к точке. Математический анализ дает нам способ найти наклон в любой отдельной точке кривой.

- Нам также необходимо понятие **предела**. Предел описывает, к какому значению приближается функция по мере того, как ее вход становится все ближе и ближе к некоторой цели, не обязательно достигая ее.

$$\lim_{x \to a} f(x) = L$$

- Это читается так: «по мере того как $x$ стремится к $a$, $f(x)$ стремится к $L$». Функции не обязательно быть равной $L$ в точке $x = a$. Ей достаточно лишь стать произвольно близкой к этому значению.

- Например, возьмем $f(x) = \frac{x^2 - 1}{x - 1}$. Если подставить $x = 1$ напрямую, получится $\frac{0}{0}$, что не определено.

- Но попробуйте значения, близкие к 1: $f(0.9) = 1.9$, $f(0.99) = 1.99$, $f(1.01) = 2.01$. Выходы явно стремятся к 2.

- Алгебраически мы можем понять почему: разложим числитель как $(x-1)(x+1)$, сократим члены $(x-1)$, и получим $f(x) = x + 1$ для всех $x \neq 1$. Таким образом, при $x \to 1$, $f(x) \to 2$.

- У функции есть «дырка» в точке $x = 1$, но предел все равно существует.

- Пределы — это фундамент, на котором строится все остальное в математическом анализе.

- **Производная** функции $f(x)$ в точке $x = a$ измеряет мгновенную скорость изменения. Геометрически это наклон касательной к кривой в этой точке.

![Производная — это наклон касательной в точке на кривой](../images/tangent_line.svg)

- Чтобы вычислить этот наклон, мы берем две точки на кривой и вычисляем наклон прямой, проходящей через них (**секущей**). Затем мы сдвигаем вторую точку все ближе к первой и смотрим, к какому наклону стремится секущая. Это и есть **разностное отношение**:

$$f'(a) = \lim_{h \to 0} \frac{f(a + h) - f(a)}{h}$$

![По мере уменьшения h секущая приближается к касательной](../images/difference_quotient.svg)

- Числитель $f(a+h) - f(a)$ — это изменение выхода. Знаменатель $h$ — это изменение входа. Их отношение — это средняя скорость изменения на крошечном интервале. По мере того как $h \to 0$, это среднее значение становится мгновенной скоростью.

- Например, пусть $f(x) = x^2$. В точке $x = 3$:

$$f'(3) = \lim_{h \to 0} \frac{(3+h)^2 - 9}{h} = \lim_{h \to 0} \frac{9 + 6h + h^2 - 9}{h} = \lim_{h \to 0} (6 + h) = 6$$

- Таким образом, в точке $x = 3$ функция $x^2$ возрастает со скоростью 6 единиц выхода на единицу входа.

- Функция является **дифференцируемой** в точке, если этот предел существует. Для этого функция должна быть непрерывной (без скачков), гладкой (без острых углов) и определенной в окрестности этой точки.

- Если вы можете нарисовать кривую, не отрывая ручки и без изломов, она, вероятно, дифференцируема в этой точке.

- Вычисление производных каждый раз по определению предела было бы утомительным. К счастью, существует несколько правил, позволяющих быстро дифференцировать почти любую функцию.

- **Правило константы**: производная константы равна нулю. Если $f(x) = 5$, то $f'(x) = 0$. У горизонтальной прямой наклон равен нулю.

- **Правило степени**: рабочая лошадка дифференцирования. Вынесите показатель степени вперед и уменьшите его на единицу:

$$\frac{d}{dx} x^n = n x^{n-1}$$

- Например: $\frac{d}{dx} x^3 = 3x^2$. Кубическая функция становится квадратичной. Это работает для любого вещественного показателя, включая отрицательные и дробные: $\frac{d}{dx} x^{-1} = -x^{-2}$ и $\frac{d}{dx} \sqrt{x} = \frac{d}{dx} x^{1/2} = \frac{1}{2}x^{-1/2}$.

- **Правило суммы/разности**: дифференцируйте по отдельности каждый член.

$$\frac{d}{dx}[f(x) \pm g(x)] = f'(x) \pm g'(x)$$

- **Правило произведения**: когда две функции перемножаются, производная — это не просто произведение производных. Вместо этого:

$$\frac{d}{dx}[f(x) \cdot g(x)] = f'(x)g(x) + f(x)g'(x)$$

- Думайте об этом так: «скорость изменения первого умножить на второе плюс первое умножить на скорость изменения второго». Например, $\frac{d}{dx}[x^2 \sin x] = 2x \sin x + x^2 \cos x$.

- **Правило частного**: для отношения функций:

$$\frac{d}{dx}\left[\frac{f(x)}{g(x)}\right] = \frac{f'(x)g(x) - f(x)g'(x)}{[g(x)]^2}$$

- Полезная мнемоника: «производная числителя на знаменатель минус числитель на производную знаменателя, делённое на квадрат знаменателя».

- **Цепное правило**: важнейшее правило для машинного обучения. Когда функции являются композицией (одна внутри другой), производная равна произведению производных вдоль цепочки:

$$\frac{d}{dx} f(g(x)) = f'(g(x)) \cdot g'(x)$$

- Представьте это как очистку луковицы. Дифференцируйте внешнюю функцию (оставляя внутреннюю без изменений), затем умножьте на производную внутренней функции.

![Chain rule: differentiate the outer, multiply by the derivative of the inner](../images/chain_rule.svg)

- Например, $\frac{d}{dx} (3x + 1)^5 = 5(3x+1)^4 \cdot 3 = 15(3x+1)^4$. Внешняя функция — $(\cdot)^5$, а внутренняя — $3x+1$.

- Цепное правило является математической основой **обратного распространения ошибки** в нейронных сетях. Глубокая сеть — это длинная цепочка композиций функций. Чтобы вычислить, как функция потерь меняется по отношению к каждому весу, мы многократно применяем цепное правило от выходного слоя обратно к входному, умножая локальные производные на каждом шаге.

- Ниже приведены наиболее распространённые производные, с которыми вы столкнётесь. Каждую из них можно вывести из определения предела, но знание их наизусть экономит время:

| Функция | Производная | Примечания |
|---|---|---|
| $e^x$ | $e^x$ | Единственная функция, равная своей производной |
| $a^x$ | $a^x \ln a$ | Обобщение экспоненты |
| $\ln x$ | $\frac{1}{x}$ | Натуральный логарифм |
| $\log_a x$ | $\frac{1}{x \ln a}$ | Общий логарифм |
| $\sin x$ | $\cos x$ | |
| $\cos x$ | $-\sin x$ | Обратите внимание на знак минус |
| $\tan x$ | $\sec^2 x$ | |

- Экспоненциальная функция $e^x$ примечательна: это единственная функция, которая равна своей производной. Именно поэтому $e$ встречается в машинном обучении повсюду: от функций активации softmax до вероятностных распределений.

- **Правило Лопиталя** позволяет вычислять пределы, приводящие к неопределённостям вида $\frac{0}{0}$ или $\frac{\infty}{\infty}$. Когда прямая подстановка даёт одну из этих форм, можно взять производную числителя и знаменателя по отдельности и снова вычислить предел:

$$\lim_{x \to a} \frac{f(x)}{g(x)} = \lim_{x \to a} \frac{f'(x)}{g'(x)}$$

- Условия: обе функции $f$ и $g$ должны быть дифференцируемы в окрестности $a$, и $g'(x) \neq 0$ в окрестности $a$ (за исключением, возможно, самой точки $a$). Исходный предел должен давать неопределённость.

- Например: $\lim_{x \to 0} \frac{\sin x}{x}$. Прямая подстановка даёт $\frac{0}{0}$. Применяя правило Лопиталя: $\lim_{x \to 0} \frac{\cos x}{1} = 1$. Этот предел фундаментален, он встречается в обработке сигналов и анализе Фурье.

- Вы можете применять правило многократно, если результат всё ещё является неопределённостью. Например, $\lim_{x \to 0} \frac{1 - \cos x}{x^2}$ даёт $\frac{0}{0}$. Первое применение: $\lim_{x \to 0} \frac{\sin x}{2x}$, всё ещё $\frac{0}{0}$. Второе применение: $\lim_{x \to 0} \frac{\cos x}{2} = \frac{1}{2}$.

- Если две функции дифференцируемы, то их сумма, разность, произведение, композиция и частное (где знаменатель не равен нулю) также дифференцируемы. Именно поэтому мы можем уверенно дифференцировать сложные выражения, построенные из простых частей.

1. Визуализируйте распространённые функции. Постройте графики $x^2$, $\sin(x)$ и $e^x$ рядом, чтобы сформировать интуитивное понимание того, как разные формулы создают разные формы. Попробуйте изменить параметры (например, $2x^2$, $\sin(2x)$) и понаблюдайте, как меняются кривые.
```python
import jax.numpy as jnp
import matplotlib.pyplot as plt

x = jnp.linspace(-3, 3, 300)

fig, axes = plt.subplots(1, 3, figsize=(12, 3))
axes[0].plot(x, x**2, color="#e74c3c")
axes[0].set_title("x²  (parabola)")
axes[1].plot(x, jnp.sin(x), color="#3498db")

axes[2].set_title("eˣ  (exponential)")
for ax in axes:
    ax.axhline(0, color="gray", linewidth=0.5)
    ax.axvline(0, color="gray", linewidth=0.5)
plt.tight_layout()
plt.show()
```

2. Используйте автоматическое дифференцирование JAX для вычисления производной $f(x) = x^3 - 2x + 1$ в нескольких точках. Сравните с аналитической производной $f'(x) = 3x^2 - 2$.
```python
import jax
import jax.numpy as jnp

f = lambda x: x**3 - 2*x + 1
df = jax.grad(f)

for x in [0.0, 1.0, 2.0, -1.0]:
    print(f"x={x:5.1f}  autodiff: {df(x):.4f}  analytical: {3*x**2 - 2:.4f}")
```

2. Проверьте цепное правило численно. Определите $f(x) = \sin(x^2)$, вычислите её производную с помощью `jax.grad` и сравните с аналитическим результатом $2x\cos(x^2)$.
```python
import jax
import jax.numpy as jnp

f = lambda x: jnp.sin(x**2)
df = jax.grad(f)

for x in [0.5, 1.0, 2.0]:
    auto = df(x)
    analytical = 2*x * jnp.cos(x**2)
    print(f"x={x:.1f}  autodiff: {auto:.6f}  analytical: {analytical:.6f}")
```

3. Визуализируйте производную. Постройте график $f(x) = x^3 - 3x$ и её производной $f'(x) = 3x^2 - 3$ на одном графике. Заметьте, как точки, где $f'(x) = 0$, соответствуют пикам и впадинам функции $f$.
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

f = lambda x: x**3 - 3*x
# jax.grad works on scalars; jax.vmap vectorises it to operate on an array of inputs at once
df = jax.vmap(jax.grad(f))

x = jnp.linspace(-2.5, 2.5, 200)
plt.plot(x, jax.vmap(f)(x), label="f(x)")
plt.plot(x, df(x), label="f'(x)", linestyle="--")
plt.axhline(0, color="gray", linewidth=0.5)
plt.legend()
plt.title("A function and its derivative")
plt.show()
```
