# 采样

*采样决定了我们如何收集数据，并直接控制我们得出的每个结论的质量。本文件涵盖随机采样、分层采样、整群采样、系统采样、采样分布、大数定律以及自助法——这些方法对 ML 中的训练/测试划分和数据集策划至关重要。*

- 在理想世界中，你会测量所关心群体的每一个成员。但在实践中这几乎不可能。你无法调查每位选民、测试每个灯泡或扫描每位病人。于是你抽取一个 **sample（样本）**，用它来了解整体。

- **population（总体）** 是你想研究的个体或物品的完整集合。**sample** 是你实际观测到的子集。

- **参数** 是描述 population 的一个数字（例如某国所有成年人真实平均身高）。

- **统计量** 是从你的 sample 计算出的一个数字（例如你测量的 500 人的平均身高）。统计量用来估计参数。

- 结论的质量完全取决于你如何选择 sample。有偏的 sample 会导致有偏的结论，无论你的分析多么精细。

- **抽样框** 是你实际从中抽取 sample 的所有个体的列表。理想情况下它应与 population 完全吻合，但实践中总有差距。

- 例如，如果你通过电话调查，就会漏掉所有没有电话的人。抽样框与 population 之间的差异称为 **覆盖误差**。

- **抽样误差** 是 sample 统计量与 population 参数之间的自然差异。

- 即使是完全随机的 sample 也不会与 population 完全一致。更大的 sample 能减小抽样误差。

- 采样有两大类：概率采样和非概率采样。

- **概率采样** 指 population 中每个成员都有已知且非零的被选机会。这使你能量化不确定性并推广结论。

- **简单随机采样**：每个个体被选中的机会相等，且每个大小为 $n$ 的可能 sample 都同等可能。想象把每个名字放进帽子里盲抽。

- **分层采样**：根据某个共同特征（如年龄段、地区）把 population 划分为不重叠的组（层），再从每层中随机采样。这保证了每个组都有代表，并在各层之间差异较大时减小 variance。

- **整群采样**：把 population 划分为组（群），随机选取部分群，再纳入所选群中的所有人。当 population 在地理上分散时很实用，比如在一个学区里整校采样而非对个别学生采样。

- **系统采样**：随机选一个起点，然后从列表中每隔第 $k$ 个个体取一个。例如从第 7 人开始，然后每隔 10 人取一个（7、17、27……）。实现简单，但如果列表有隐藏模式，可能引入偏差。

![三种概率采样方法并排：简单随机、分层、整群](../images/sampling_methods.svg)

- **非概率采样** 不给每个成员已知的被选机会。结论无法严格推广，但这些方法通常更快更便宜。

- **方便采样**：选最容易接触的人。在购物中心调查人很方便，但会漏掉不在那里购物的人。

- **配额采样**：类似分层采样，但没有随机性。研究者从每组中挑选可接触的个体来填满配额（例如 50 名男和 50 名女）。

- **滚雪球采样**：从少数参与者开始，请他们招募其他人。适用于难以接触的群体（例如研究罕见病），但严重偏向有社交联系的个体。

- 一旦有了采样方法，自然产生一个问题：如果我取另一个 sample，会得到不同的统计量吗？几乎肯定。**采样分布** 是某统计量（如 sample 均值）在所有同等大小的可能 sample 上的分布。

- 想象抽取 1,000 个各 30 人的不同 sample，并计算每个的均值身高。这 1,000 个均值形成一个 distribution。有些会略高于真实 population 均值，有些略低，大多数聚集在真值附近。

- 该采样分布的 standard deviation 称为 **标准误**：

$$SE = \frac{\sigma}{\sqrt{n}}$$

- 注意标准误随 $n$ 增大而缩小。更大的 sample 给出更精确的估计。样本量翻四倍，标准误减半。

- 统计学中最重要的结果是 **中心极限定理（CLT）**。它说：无论原始 population 的形状如何，当样本量增大时，sample 均值的分布趋近于正态分布。

![CLT：偏斜的 population 产生正态分布的 sample 均值](../images/central_limit_theorem.svg)

- 更精确地说，若 $X_1, X_2, \ldots, X_n$ 是来自任意 distribution 的独立观测，该 distribution 的均值为 $\mu$、variance 为有限值 $\sigma^2$，则随着 $n$ 增大：

$$\bar{X} \approx \text{Normal}\!\left(\mu, \frac{\sigma^2}{n}\right)$$

- CLT 是大部分推断性统计能起作用的原因。它使我们能在底层数据非正态时，只要 sample 足够大，仍用正态分布作近似。

- "足够大"是多大？一个常用的经验法则是 $n \ge 30$，但这取决于 population 的非正态程度。对于高度偏斜的 distribution，可能需要更多。对于大致对称的 population，即使 $n = 10$ 也可能足够。

- CLT 有三个关键条件：
    - **独立性**：每个观测不影响其他观测
    - **有限方差**：population 的 variance 必须存在（排除一些奇异 distribution）
    - **同分布**：所有观测来自同一 distribution

1. 可视化演示 CLT：从高度偏斜的 distribution 中抽取样本，计算 sample 均值，观察均值直方图趋于钟形。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

key = jax.random.PRNGKey(0)

# Exponential distribution (very skewed)
population = jax.random.exponential(key, shape=(100_000,))

fig, axes = plt.subplots(1, 4, figsize=(14, 3))
sample_sizes = [1, 5, 30, 100]

for ax, n in zip(axes, sample_sizes):
    keys = jax.random.split(key, 2000)
    means = jnp.array([jax.random.choice(k, population, shape=(n,)).mean() for k in keys])
    ax.hist(means, bins=40, color="#3498db", alpha=0.7, density=True)
    ax.set_title(f"n = {n}")
    ax.set_xlim(0, 4)

fig.suptitle("CLT: sample means become normal as n increases", fontsize=13)
plt.tight_layout()
plt.show()
```

2. 比较简单随机采样与分层采样。构造一个有明显分组的 population，展示分层采样在估计上给出更低 variance。
```python
import jax
import jax.numpy as jnp

key = jax.random.PRNGKey(42)

# Population: two distinct groups
group_a = jax.random.normal(key, shape=(500,)) + 10   # mean ~10
key, subkey = jax.random.split(key)
group_b = jax.random.normal(subkey, shape=(500,)) + 20  # mean ~20
population = jnp.concatenate([group_a, group_b])

# Simple random sampling: 1000 trials, sample size 20
srs_means = []
for i in range(1000):
    key, subkey = jax.random.split(key)
    sample = jax.random.choice(subkey, population, shape=(20,), replace=False)
    srs_means.append(sample.mean())
srs_means = jnp.array(srs_means)

# Stratified sampling: 10 from each group
strat_means = []
for i in range(1000):
    key, k1, k2 = jax.random.split(key, 3)
    s_a = jax.random.choice(k1, group_a, shape=(10,), replace=False)
    s_b = jax.random.choice(k2, group_b, shape=(10,), replace=False)
    strat_means.append(jnp.concatenate([s_a, s_b]).mean())
strat_means = jnp.array(strat_means)

print(f"Simple Random - Mean: {srs_means.mean():.3f}, Std: {srs_means.std():.3f}")
print(f"Stratified    - Mean: {strat_means.mean():.3f}, Std: {strat_means.std():.3f}")
print(f"Stratified sampling reduced variance by {(1 - strat_means.var()/srs_means.var())*100:.1f}%")
```

3. 探究样本量如何影响标准误。绘制标准误随样本量的变化，确认 $1/\sqrt{n}$ 关系。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

key = jax.random.PRNGKey(7)
population = jax.random.normal(key, shape=(50_000,)) * 10 + 50

sample_sizes = [5, 10, 20, 50, 100, 200, 500, 1000]
std_errors = []

for n in sample_sizes:
    means = []
    for _ in range(500):
        key, subkey = jax.random.split(key)
        sample = jax.random.choice(subkey, population, shape=(n,))
        means.append(sample.mean())
    std_errors.append(jnp.array(means).std())

plt.figure(figsize=(8, 4))
plt.plot(sample_sizes, std_errors, "o-", color="#e74c3c", label="Observed SE")
theoretical = population.std() / jnp.sqrt(jnp.array(sample_sizes, dtype=jnp.float32))
plt.plot(sample_sizes, theoretical, "--", color="#3498db", label="σ/√n (theoretical)")
plt.xlabel("Sample size (n)")
plt.ylabel("Standard error")
plt.legend()
plt.title("Standard error shrinks with larger samples")
plt.show()
```
