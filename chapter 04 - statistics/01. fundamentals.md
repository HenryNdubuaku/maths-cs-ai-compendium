# Основы статистики

*Статистика предоставляет язык для описания данных и количественной оценки неопределенности. В этом файле рассматриваются распределения, случайные величины, функции вероятности (PMF), функции плотности вероятности (PDF), функции распределения (CDF), математическое ожидание, дисперсия, моменты и центральная предельная теорема — концепции, лежащие в основе каждой метрики оценки и функции потерь в машинном обучении.*

- Статистика — это наука об извлечении знаний из данных. Вы собираете наблюдения, обобщаете их и делаете выводы, часто о вещах, которые невозможно измерить напрямую.

- Представьте, что вы хотите узнать средний рост каждого взрослого человека в стране. Вы не можете измерить всех, поэтому вы измеряете выборку и используете статистику, чтобы сделать обоснованное предположение обо всей совокупности.

- Существует два основных раздела:
    - **Описательная статистика**: обобщение уже имеющихся данных (средние значения, графики, таблицы)
    - **Математическая статистика (инференция)**: использование выборки для утверждений о более широкой группе

- Фундаментальным элементом статистики является **распределение** — описание того, как распределены значения. Все остальное — средние значения, тесты, прогнозы — вытекает из понимания распределений.

- **Частотное распределение** подсчитывает, как часто каждое значение (или диапазон значений) встречается в ваших данных. Представьте, что вы сортируете результаты экзаменов по корзинам и считаете, сколько студентов попало в каждую из них. Результатом будет гистограмма.

- **Вероятностное распределение** заменяет необработанные счетчики вероятностями. Вместо «12 студентов набрали от 70 до 80 баллов» оно говорит: «вероятность набрать от 70 до 80 баллов составляет 0.24». Столбцы гистограммы превращаются в плавную кривую, когда данные непрерывны.

![Частотное распределение в виде гистограммы против вероятностного распределения в виде плавной кривой](../images/distribution_types.svg)

- Гистограмма слева построена на основе реальных данных, которые вы собрали. Плавная кривая справа — это математическая модель, описывающая закономерность, лежащую в основе данных. Одно является эмпирическим, другое — теоретическим.

- Чтобы работать с распределениями математически, нам нужен способ сопоставить числа с исходами. Именно это и делает **случайная величина**.

- Случайная величина — это функция, которая отображает каждый исход эксперимента в вещественное число. Подбросьте монету: исход — «орел» или «решка», но случайная величина $X$ преобразует это в $X(\text{heads}) = 1$ и $X(\text{tails}) = 0$. Теперь мы можем выполнять арифметические действия.

![Случайная величина, отображающая исходы (монета, кость) на числовую прямую](../images/random_variable.svg)

- **Дискретная** случайная величина принимает счетное множество значений: количество орлов при 10 подбрасываниях, результат броска кости, количество электронных писем, которые вы получаете за час.

- **Непрерывная** случайная величина может принимать любое значение в интервале: ваш точный рост, время до прибытия следующего автобуса, температура в полдень.

- Это различие важно, потому что оно меняет способ вычисления вероятностей. Для дискретных величин мы суммируем. Для непрерывных величин мы интегрируем (вспомните интегралы из главы 3).

- Для дискретной случайной величины **функция вероятности (PMF)** дает вероятность каждого конкретного значения:

$$P(X = x) = p(x), \quad \text{where } \sum_{x} p(x) = 1$$

- Для непрерывной случайной величины **функция плотности вероятности (PDF)** дает вероятность попадания в диапазон. Вероятность любого отдельного точного значения равна нулю; только интервалы имеют положительную вероятность:

$$P(a \le X \le b) = \int_a^b f(x)\, dx, \quad \text{where } \int_{-\infty}^{\infty} f(x)\, dx = 1$$

- Теперь, когда мы можем присваивать числа исходам, самый естественный вопрос: какое значение мы ожидаем в среднем?

- **Математическое ожидание** (или ожидаемое значение) — это средневзвешенное всех возможных значений, где весами являются вероятности. Думайте об этом как о «центре тяжести» распределения.

- Если вы много раз бросите честную игральную кость, ваш средний результат будет стремиться к 3.5. Это и есть математическое ожидание, даже если вы никогда не сможете фактически выбросить 3.5.

- Для дискретной случайной величины:

$$E[X] = \sum_{x} x \cdot p(x)$$

- Для непрерывной случайной величины (используя интеграл из главы 3):

$$E[X] = \int_{-\infty}^{\infty} x \cdot f(x)\, dx$$

- Пример: у честной шестигранной кости $p(x) = 1/6$ для $x = 1, 2, 3, 4, 5, 6$.

$$E[X] = 1 \cdot \tfrac{1}{6} + 2 \cdot \tfrac{1}{6} + 3 \cdot \tfrac{1}{6} + 4 \cdot \tfrac{1}{6} + 5 \cdot \tfrac{1}{6} + 6 \cdot \tfrac{1}{6} = \frac{21}{6} = 3.5$$

- Математическое ожидание линейно, что означает $E[aX + b] = aE[X] + b$. Это свойство чрезвычайно полезно и постоянно встречается в функциях потерь машинного обучения.

- Математическое ожидание указывает на центр, но ничего не говорит о том, насколько разбросаны значения. Чтобы описать полную форму распределения, нам нужны **моменты**.

- Момент — это математическое ожидание степени $X$. $k$-й **начальный момент** равен:

$$\mu_k' = E[X^k]$$

- Первый начальный момент ($k = 1$) — это просто среднее значение: $\mu_1' = E[X] = \mu$.

- Начальные моменты измеряются от нуля. Часто нас интересует отклонение от среднего. $k$-й **центральный момент** центрирует измерение:

$$\mu_k = E[(X - \mu)^k]$$

- Первый центральный момент всегда равен нулю (отклонения выше и ниже среднего взаимно уничтожаются). Второй центральный момент — это **дисперсия**.

- Чтобы сравнивать распределения в разных масштабах, мы проводим **стандартизацию**, деля на соответствующую степень стандартного отклонения $\sigma$:

$$\tilde{\mu}_k = \frac{\mu_k}{\sigma^k}$$

- Каждый момент отражает разный аспект формы распределения:

![Колоколообразная кривая с аннотациями того, что отражает каждый момент: среднее (центр), дисперсия (разброс), асимметрия (скошенность), эксцесс (вес хвостов)](../images/moments_shape.svg)

- **1-й момент (Среднее)**: где центрировано распределение. Точка баланса.
- **2-й момент (Дисперсия)**: насколько значения разбросаны вокруг среднего. Большая дисперсия означает большую ширину.
- **3-й момент (Асимметрия)**: наклонено ли распределение влево или вправо. Нулевая асимметрия означает симметрию.
- **4-й момент (Эксцесс)**: насколько тяжелы хвосты. Более высокий эксцесс означает наличие более экстремальных выбросов.

- Давайте вычислим все четыре момента для конкретного набора данных: $X = \{2, 4, 4, 4, 5, 5, 7, 9\}$.

- **Шаг 1: Среднее** (1-й начальный момент)

$$\mu = \frac{2 + 4 + 4 + 4 + 5 + 5 + 7 + 9}{8} = \frac{40}{8} = 5$$

- **Шаг 2: Дисперсия** (2-й центральный момент). Вычтите среднее из каждого значения, возведите в квадрат, затем найдите среднее:

$$\sigma^2 = \frac{(2{-}5)^2 + (4{-}5)^2 + (4{-}5)^2 + (4{-}5)^2 + (5{-}5)^2 + (5{-}5)^2 + (7{-}5)^2 + (9{-}5)^2}{8}$$

$$= \frac{9 + 1 + 1 + 1 + 0 + 0 + 4 + 16}{8} = \frac{32}{8} = 4$$

- **Стандартное отклонение** равно $\sigma = \sqrt{4} = 2$.

- **Шаг 3: Асимметрия** (нормированный 3-й центральный момент). Возведите отклонения в куб, найдите среднее, разделите на $\sigma^3$:

$$\tilde{\mu}_3 = \frac{1}{8} \cdot \frac{(-3)^3 + (-1)^3 + (-1)^3 + (-1)^3 + 0^3 + 0^3 + 2^3 + 4^3}{2^3}$$

$$= \frac{1}{8} \cdot \frac{-27 -1 -1 -1 + 0 + 0 + 8 + 64}{8} = \frac{42}{64} = 0.656$$

- Положительная асимметрия означает, что правый хвост длиннее, что логично, так как 9 значительно превышает среднее значение.

- **Шаг 4: Эксцесс** (нормированный 4-й центральный момент). Возведите отклонения в 4-ю степень:

$$\tilde{\mu}_4 = \frac{1}{8} \cdot \frac{(-3)^4 + (-1)^4 + (-1)^4 + (-1)^4 + 0^4 + 0^4 + 2^4 + 4^4}{2^4}$$

$$= \frac{1}{8} \cdot \frac{81 + 1 + 1 + 1 + 0 + 0 + 16 + 256}{16} = \frac{356}{128} = 2.781$$

- Нормальное распределение имеет эксцесс, равный 3 (называется «мезокуртическим»). Наше значение 2.781 близко к этому, что говорит о том, что хвосты распределения примерно нормальные. Значения выше 3 («лептокуртические») указывают на более тяжелые хвосты; значения ниже 3 («платикуртические») — на более легкие. Некоторые формулы используют **избыточный эксцесс**, вычитая 3, поэтому наш избыточный эксцесс составил бы $-0.219$.

## Задачи по программированию (используйте CoLab или ноутбук)

1. Вычислите математическое ожидание для «заряженной» игральной кости, где грань 6 выпадает с вероятностью 0.3, а остальные грани делят оставшуюся вероятность поровну. Проверьте результат, симулировав 100 000 бросков.
```python
import jax
import jax.numpy as jnp

# Loaded die: face 6 has p=0.3, others share 0.7 equally
probs = jnp.array([0.14, 0.14, 0.14, 0.14, 0.14, 0.30])
faces = jnp.array([1, 2, 3, 4, 5, 6])

# Analytical expected value
ev = jnp.sum(faces * probs)
print(f"Expected value (formula): {ev:.4f}")

# Simulation
key = jax.random.PRNGKey(42)
rolls = jax.random.choice(key, faces, shape=(100_000,), p=probs)
print(f"Expected value (simulation): {rolls.mean():.4f}")
```

2. Вычислите все четыре момента (среднее, дисперсию, асимметрию, эксцесс) для датасета из разобранного примера, затем измените данные и пронаблюдайте, как меняется каждый момент.
```python
import jax.numpy as jnp

x = jnp.array([2, 4, 4, 4, 5, 5, 7, 9], dtype=jnp.float32)

mean = jnp.mean(x)
variance = jnp.mean((x - mean) ** 2)
std = jnp.sqrt(variance)
skewness = jnp.mean(((x - mean) / std) ** 3)
kurtosis = jnp.mean(((x - mean) / std) ** 4)

print(f"Mean:     {mean:.3f}")
print(f"Variance: {variance:.3f}")
print(f"Std Dev:  {std:.3f}")
print(f"Skewness: {skewness:.3f}")
print(f"Kurtosis: {kurtosis:.3f}")
print(f"Excess K: {kurtosis - 3:.3f}")
```

3. Визуализируйте функцию вероятности (PMF) и функцию распределения (CDF) для честной игральной кости рядом друг с другом. Попробуйте изменить вероятности, чтобы увидеть, как меняются формы графиков.
```python
import jax.numpy as jnp
import matplotlib.pyplot as plt

faces = jnp.array([1, 2, 3, 4, 5, 6])
pmf = jnp.ones(6) / 6  # fair die; try changing these!
cdf = jnp.cumsum(pmf)

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(10, 4))

ax1.bar(faces, pmf, color="#3498db", alpha=0.8)
ax1.set_title("PMF")
ax1.set_xlabel("Face")
ax1.set_ylabel("P(X = x)")
ax1.set_ylim(0, 0.5)

ax2.step(faces, cdf, where="mid", color="#e74c3c", linewidth=2)
ax2.set_title("CDF")
ax2.set_xlabel("Face")
ax2.set_ylabel("P(X ≤ x)")
ax2.set_ylim(0, 1.1)

plt.tight_layout()
plt.show()
```
