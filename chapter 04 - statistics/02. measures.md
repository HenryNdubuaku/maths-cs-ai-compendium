# Статистические показатели

*Статистические показатели обобщают данные с помощью отдельных чисел, которые отражают разброс, положение, форму и взаимосвязь. В этом файле рассматриваются дисперсия, стандартное отклонение, квартили, асимметрия, эксцесс, ковариация, корреляция и z-оценки — инструментарий для разведочного анализа данных и проектирования признаков в машинном обучении.*

- В предыдущем файле мы представили моменты как семейство сводных статистик. Здесь мы разберем практические инструменты, которые из них вытекают: показатели рассеяния, положения, формы и взаимосвязи.

- **Рассеяние** отвечает на вопрос: насколько сильно разбросаны данные? У двух классов может быть одинаковый средний балл за тест, но совершенно разный разброс.

![Два распределения с одинаковым средним, но разным разбросом](../images/variance_spread.svg)

- У узкого (синего) распределения низкая дисперсия: большинство значений плотно группируются вокруг среднего. У широкого (красного) распределения высокая дисперсия: значения рассеяны дальше.

- **Дисперсия** — это среднее квадратичное отклонение от среднего значения. Мы возводим в квадрат, чтобы положительные и отрицательные отклонения не компенсировали друг друга.

$$\sigma^2 = \frac{1}{N} \sum_{i=1}^{N} (x_i - \mu)^2$$

- При работе с выборкой (а не с генеральной совокупностью) мы делим на $N - 1$ вместо $N$. Эта поправка (называемая поправкой Бесселя) учитывает тот факт, что выборка имеет тенденцию недооценивать истинную изменчивость:

$$s^2 = \frac{1}{N-1} \sum_{i=1}^{N} (x_i - \bar{x})^2$$

- **Стандартное отклонение** — это квадратный корень из дисперсии: $\sigma = \sqrt{\sigma^2}$. Оно возвращает показатель к исходным единицам измерения. Если ваши данные в сантиметрах, дисперсия будет в см$^2$, а стандартное отклонение — снова в см.

- **Среднее абсолютное отклонение (MAD)** — это более простая альтернатива. Вместо возведения в квадрат берется абсолютное значение каждого отклонения:

$$\text{MAD} = \frac{1}{N} \sum_{i=1}^{N} |x_i - \mu|$$

- MAD более устойчиво к выбросам, чем дисперсия, поскольку не усиливает большие отклонения путем возведения в квадрат. Однако дисперсия математически удобнее (она хорошо разлагается в доказательствах и при оптимизации в машинном обучении).

- **Положение** отвечает на другой вопрос: где конкретное значение находится относительно остальных данных?

- **Квартили** делят отсортированные данные на четыре равные части. Q1 (25-й перцентиль) — это значение, ниже которого находится 25% данных. Q2 — это медиана (50-й перцентиль). Q3 — это 75-й перцентиль.

- **Межквартильный размах (IQR)** равен $Q3 - Q1$. Он отражает разброс средних 50% данных, игнорируя экстремальные значения.

![Ящичковая диаграмма, показывающая Q1, медиану, Q3, IQR, «усы» и выброс](../images/quartiles_boxplot.svg)

- Ящичковая диаграмма (box plot) — одна из самых полезных визуализаций в статистике. Ящик охватывает диапазон от Q1 до Q3, линия внутри — это медиана, «усы» простираются до наиболее экстремальных значений, не являющихся выбросами, а точки за пределами «усов» — это выбросы.

- **Перцентили** обобщают квартили. $p$-й перцентиль — это значение, ниже которого находится $p\%$ наблюдений. Q1 — это 25-й перцентиль, медиана — 50-й, а Q3 — 75-й.

- **z-оценка** показывает, на сколько стандартных отклонений значение отстоит от среднего:

$$z = \frac{x - \mu}{\sigma}$$

- z-оценка, равная 2, означает, что значение находится на 2 стандартных отклонения выше среднего. z-оценка $-1.5$ означает, что оно на 1.5 стандартных отклонения ниже. Это также называется **стандартизацией** и широко используется в машинном обучении для масштабирования признаков, так как преобразует любое распределение к среднему 0 и стандартному отклонению 1.

- **Форма** описывает геометрию распределения помимо его центра и разброса.

- **Асимметрия** (стандартизированный 3-й момент из предыдущего файла) измеряет несимметричность. У идеально симметричного распределения, такого как нормальная кривая, асимметрия равна нулю. Положительная асимметрия означает более длинный правый хвост (например, распределение доходов). Отрицательная асимметрия означает более длинный левый хвост (например, возраст выхода на пенсию).

$$\text{Skewness} = \frac{1}{N} \sum_{i=1}^{N} \left(\frac{x_i - \mu}{\sigma}\right)^3$$

- **Эксцесс** (стандартизированный 4-й момент) измеряет «тяжесть» хвостов. У нормального распределения эксцесс равен 3. Распределения с более тяжелыми хвостами (более склонные к выбросам) имеют эксцесс больше 3.

$$\text{Kurtosis} = \frac{1}{N} \sum_{i=1}^{N} \left(\frac{x_i - \mu}{\sigma}\right)^4$$

- **Корреляция** измеряет силу и направление связи между двумя переменными. Она отвечает на вопрос: когда одна переменная растет, имеет ли другая тенденцию расти, убывать или не меняться?

![Три диаграммы рассеяния, показывающие положительную, нулевую и отрицательную корреляцию](../images/correlation_scatter.svg)

- **Корреляция Пирсона** ($r$) измеряет *линейную* связь. Она варьируется от $-1$ (идеальная отрицательная) через $0$ (отсутствие связи) до $+1$ (идеальная положительная).

$$r = \frac{\sum_{i=1}^{N} (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum (x_i - \bar{x})^2} \cdot \sqrt{\sum (y_i - \bar{y})^2}}$$

- Если вы помните скалярное произведение из главы 1, то корреляция Пирсона — это, по сути, косинусное сходство между центрированными относительно среднего версиями векторов $\mathbf{x}$ и $\mathbf{y}$.

- **Корреляция Спирмена** ($\rho$) измеряет *монотонную* связь. Вместо использования исходных значений она сначала ранжирует их, а затем вычисляет корреляцию Пирсона для этих рангов. Это делает ее устойчивой к выбросам и позволяет работать даже при нелинейной связи, если она монотонно возрастает или убывает.

- **Среднее геометрическое** — это подходящее среднее значение, когда величины перемножаются, например, темпы роста. Если ваши инвестиции растут на 10%, затем на 20%, затем на 30%, средний коэффициент роста не является средним арифметическим этих ставок. Вместо этого:

$$\bar{x}_{\text{geo}} = \left(\prod_{i=1}^{N} x_i\right)^{1/N}$$

- Для темпов роста сначала переведите проценты в коэффициенты (1.10, 1.20, 1.30), вычислите среднее геометрическое, а затем вычтите 1.

- **Экспоненциальное скользящее среднее (EMA)** придает больший вес недавним наблюдениям. В отличие от простого скользящего среднего, где все точки в окне имеют равный вес, EMA убывает экспоненциально:

$$\text{EMA}_t = \alpha \cdot x_t + (1 - \alpha) \cdot \text{EMA}_{t-1}$$

- Коэффициент сглаживания $\alpha$ (от 0 до 1) определяет, как быстро старые наблюдения теряют свое влияние. Более высокое $\alpha$ означает большую чувствительность к недавним изменениям, более низкое $\alpha$ — более плавный график. В машинном обучении EMA используется в оптимизаторах, таких как Adam, и в статистике батч-нормализации.

- **Детекция выбросов** (outlier detection) позволяет выявить точки данных, которые необычно сильно отклоняются от остальных. Два распространенных метода:
    - **Метод межквартильного размаха (IQR)**: точка считается выбросом, если она лежит ниже $Q1 - 1.5 \times \text{IQR}$ или выше $Q3 + 1.5 \times \text{IQR}$.
    - **Метод Z-оценки (Z-score)**: точка считается выбросом, если $|z| > 3$ (более чем на 3 стандартных отклонения от среднего).

- Метод IQR является более устойчивым (робастным), так как он не предполагает нормального распределения. Метод Z-оценки хорошо работает, когда данные распределены приблизительно нормально, но может давать сбои, если распределение сильно скошено.

1. Вычислите дисперсию, стандартное отклонение и среднее абсолютное отклонение (MAD) для датасета и сравните их. Понаблюдайте, что произойдет при добавлении экстремального выброса.
```python
import jax.numpy as jnp

data = jnp.array([4, 8, 6, 5, 3, 7, 9, 5, 6, 7], dtype=jnp.float32)

mean = jnp.mean(data)
variance = jnp.var(data)
std = jnp.std(data)
mad = jnp.mean(jnp.abs(data - mean))

print("Original data:")
print(f"  Variance: {variance:.3f}, Std: {std:.3f}, MAD: {mad:.3f}")

# Add an outlier and recompute
data_outlier = jnp.append(data, 100.0)
mean2 = jnp.mean(data_outlier)
print(f"\nWith outlier (100):")
print(f"  Variance: {jnp.var(data_outlier):.3f}, Std: {jnp.std(data_outlier):.3f}, MAD: {jnp.mean(jnp.abs(data_outlier - mean2)):.3f}")
```

2. Вычислите корреляцию Пирсона и Спирмена между двумя переменными. Поэкспериментируйте с различными зависимостями.
```python
import jax
import jax.numpy as jnp

# Perfect linear relationship
x = jnp.array([1, 2, 3, 4, 5, 6, 7, 8], dtype=jnp.float32)
y = 2 * x + 1  # try changing this!

def pearson(a, b):
    a_c = a - jnp.mean(a)
    b_c = b - jnp.mean(b)
    return jnp.sum(a_c * b_c) / (jnp.sqrt(jnp.sum(a_c**2)) * jnp.sqrt(jnp.sum(b_c**2)))

def spearman(a, b):
    rank_a = jnp.argsort(jnp.argsort(a)).astype(jnp.float32)
    rank_b = jnp.argsort(jnp.argsort(b)).astype(jnp.float32)
    return pearson(rank_a, rank_b)

print(f"Pearson r:  {pearson(x, y):.4f}")
print(f"Spearman ρ: {spearman(x, y):.4f}")
```

3. Реализуйте детекцию выбросов с помощью методов IQR и Z-оценки, а затем сравните их результаты на скошенных данных.
```python
import jax.numpy as jnp

data = jnp.array([2, 3, 3, 4, 5, 5, 5, 6, 6, 7, 50], dtype=jnp.float32)

# IQR method
q1, q3 = jnp.percentile(data, 25), jnp.percentile(data, 75)
iqr = q3 - q1
lower, upper = q1 - 1.5 * iqr, q3 + 1.5 * iqr
iqr_outliers = data[(data < lower) | (data > upper)]
print(f"IQR bounds: [{lower:.1f}, {upper:.1f}]")
print(f"IQR outliers: {iqr_outliers}")

# Z-score method
z_scores = (data - jnp.mean(data)) / jnp.std(data)
z_outliers = data[jnp.abs(z_scores) > 3]
print(f"\nZ-scores: {z_scores}")
print(f"Z-score outliers (|z| > 3): {z_outliers}")
```

4. Вычислите и постройте график экспоненциального скользящего среднего (EMA) с различными коэффициентами сглаживания для зашумленных данных.
```python
import jax.numpy as jnp
import matplotlib.pyplot as plt

# Generate noisy data
key = __import__("jax").random.PRNGKey(0)
noise = __import__("jax").random.normal(key, shape=(50,))
signal = jnp.linspace(0, 5, 50) + noise

def ema(data, alpha):
    result = jnp.zeros_like(data)
    result = result.at[0].set(data[0])
    for t in range(1, len(data)):
        result = result.at[t].set(alpha * data[t] + (1 - alpha) * result[t - 1])
    return result

plt.figure(figsize=(10, 4))
plt.plot(signal, "o", alpha=0.3, label="raw data", color="#999")
for alpha, color in [(0.1, "#e74c3c"), (0.3, "#3498db"), (0.7, "#27ae60")]:
    plt.plot(ema(signal, alpha), label=f"α={alpha}", color=color, linewidth=2)
plt.legend()
plt.title("EMA with different smoothing factors")
plt.show()
```
