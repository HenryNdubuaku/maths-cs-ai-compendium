# 深度学习

*深度学习通过堆叠非线性层来构建分层表示，将原始输入自动转化为有用的特征。本文件涵盖 MLP、激活函数、反向传播、CNN、RNN、LSTM、attention、transformer、GAN、VAE、扩散模型以及归一化技术*

- 是什么让一个网络变得“深”？浅层网络只有一个隐藏层；深层网络则有许多层。深度使网络能够构建分层表示，早期层学习简单特征（边缘、色调），后期层则将它们组合成复杂概念（人脸、句子）。这种组合性正是深度学习强大能力的来源。

- 最简单的深度网络是 **多层感知机（multi-layer perceptron, MLP）**，也称为全连接网络或稠密网络。每一层计算：

$$h = \sigma(Wx + b)$$

- 这里 $W$ 是权重矩阵（第 02 章），$b$ 是偏置向量，$\sigma$ 是非线性激活函数。一层的输出会成为下一层的输入。如果没有非线性，堆叠再多层也是徒劳：$W_2(W_1 x) = (W_2 W_1)x$，这不过是另一个线性变换。这正是第 02 章中提到的矩阵乘法坍缩现象。

- **激活函数**引入了让深度有意义的非线性。

- **ReLU**（Rectified Linear Unit）：$\text{ReLU}(x) = \max(0, x)$。它是使用最广泛的激活函数。计算速度快，对正输入不会饱和，并且产生稀疏激活（许多神经元输出恰为零）。缺点是：输入为负的神经元总是输出零，如果永久停留在这种状态，它们就会“死亡”并停止学习。

- **Sigmoid**：$\sigma(x) = \frac{1}{1+e^{-x}}$，将输入压缩到 $(0, 1)$。适用于二分类的输出层，但在隐藏层中会有问题，因为当输入远离零时梯度会消失（曲线几乎是平坦的）。

- **Tanh**：$\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$，压缩到 $(-1, 1)$。零中心化（不同于 sigmoid），有利于梯度流动，但在极端值处仍会出现梯度消失。

- **GELU**（Gaussian Error Linear Unit）：$\text{GELU}(x) = x \cdot \Phi(x)$，其中 $\Phi$ 是标准正态分布的 CDF。它是 ReLU 的平滑近似，允许少量负值通过。GELU 是 GPT 和 BERT 的默认激活函数。

- **Swish**：$\text{Swish}(x) = x \cdot \sigma(x)$，另一种平滑门控。实践中表现与 GELU 类似。

![ReLU、Sigmoid、Tanh 和 GELU 的并排图，标注了各自的关键性质](../images/activation_functions.svg)

- 一个有 $d_{\text{in}}$ 个输入和 $d_{\text{out}}$ 个输出的稠密层有 $d_{\text{in}} \times d_{\text{out}} + d_{\text{out}}$ 个参数（权重加偏置）。矩阵乘法 $Wx$ 就是第 02 章中的矩阵-向量乘法。在 batch 设置中，输入是形状为 $(B, d_{\text{in}})$ 的矩阵 $X$，输出是形状为 $(B, d_{\text{out}})$ 的 $XW^T + b$。

- **通用近似定理**指出，一个具有足够多神经元的单隐藏层网络可以把紧致域上的任何连续函数近似到任意精度。听起来深度似乎无关紧要，但关键在于“足够多神经元”。在实践中，深层网络可以用比浅层网络指数级更少的参数来表示同样的函数。深度带来的是效率，而不仅仅是表达能力。

- 随着网络变深，会出现两种梯度病态。**梯度消失（vanishing gradients）**：当梯度穿过许多层时（通过链式法则，第 03 章），会被乘以许多因子。如果这些因子始终小于 1（sigmoid 和 tanh 饱和时就是这样），梯度会指数级地缩小到零，早期层几乎学不到东西。**梯度爆炸（exploding gradients）**：如果因子始终大于 1，梯度会指数级增长，导致数值溢出和训练不稳定。

- 缓解梯度消失/爆炸的方法：
  - 使用 ReLU 或 GELU 激活函数（对正输入梯度为 1，不饱和）
  - 谨慎的权重初始化
  - 归一化层
  - 残差连接（跳跃连接）
  - 梯度裁剪（针对梯度爆炸）：将梯度范数限制在最大值

- **权重初始化**很重要，因为它决定了训练开始时激活值和梯度的尺度。权重过大，激活会爆炸；过小，则会消失。

- **Xavier（Glorot）初始化**从一个方差为 $\frac{2}{d_{\text{in}} + d_{\text{out}}}$ 的分布中采样权重。它使各层的激活方差大致保持不变，前提是使用线性或 tanh 激活。

- **He（Kaiming）初始化**使用方差 $\frac{2}{d_{\text{in}}}$，针对 ReLU 激活校准（由于 ReLU 把一半激活置零，需要双倍的方差来补偿）。

- **归一化层**通过保证每层输入具有一致的统计量（大致零均值、单位方差）来稳定训练。

- **批归一化（Batch Normalisation, BatchNorm）**沿 batch 维度归一化：对每个通道/特征，计算 mini-batch 中所有样本的均值和方差，然后归一化。它添加了可学习的缩放（$\gamma$）和偏移（$\beta$）参数，使网络可以在需要时撤销归一化：

$$\hat{x} = \frac{x - \mu_B}{\sqrt{\sigma_B^2 + \epsilon}}, \quad y = \gamma \hat{x} + \beta$$

- BatchNorm 有个问题：它依赖 batch 大小。batch 很小时统计量噪声大。推理时使用运行平均而非 batch 统计量，这造成了训练/测试的不一致。

- **层归一化（Layer Normalisation, LayerNorm）**对每个样本沿特征维度归一化。它不依赖 batch 中的其他样本，因此成为 transformer 和循环网络的标准选择。

- **实例归一化（Instance Normalisation）**对每个样本和每个通道沿空间维度独立归一化。在风格迁移中很流行。

- **组归一化（Group Normalisation）**把通道分成若干组并在每组内归一化。它是 LayerNorm 和 InstanceNorm 之间的折中。

![三维张量，用彩色切片显示 BatchNorm、LayerNorm 和 InstanceNorm 分别沿哪些维度归一化](../images/normalization_types.svg)

- **Dropout** 是一种正则化技术，在训练时随机将比例为 $p$ 的神经元置零。这迫使网络不依赖任何单个神经元，鼓励冗余表示。测试时所有神经元都激活。**反向 dropout（inverted dropout）**在训练时将激活乘以 $\frac{1}{1-p}$，这样测试时无需再缩放。这是标准实现。

- **卷积神经网络（Convolutional Neural Networks, CNN）**利用空间结构。卷积层不像稠密层那样把每个输入都连到每个输出，而是在输入上滑动一个小滤波器（kernel），在每个位置计算点积。同一滤波器的权重在所有位置共享，这大幅减少了参数，并内置了平移不变性。

- **卷积运算**，对于 2D 输入和大小为 $k \times k$ 的滤波器 $K$：

$$(\text{input} * K)[i,j] = \sum_{m=0}^{k-1} \sum_{n=0}^{k-1} \text{input}[i+m, j+n] \cdot K[m, n]$$

![输入网格上滑动一个 3x3 滤波器，在每个位置进行逐元素相乘并求和，产生输出特征图](../images/cnn_convolution.svg)

- 输出大小取决于三个超参数。**步长（Stride）**控制滤波器在位置之间移动多少像素（步长 2 使空间维度减半）。**填充（Padding）**在输入边界外补零（“same” 填充保持空间尺寸，“valid” 填充则不保持）。输出大小公式为：$\text{out} = \lfloor (\text{in} - k + 2p) / s \rfloor + 1$。

- **池化（Pooling）**层对特征图下采样。最大池化取每个窗口内的最大值，平均池化取均值。池化在保留最重要信息的同时减小空间维度。

- **空洞卷积（Dilated convolutions）**在滤波器元素之间插入间隔，在不增加参数的情况下扩大感受野。膨胀率为 2 意味着 3x3 滤波器覆盖 5x5 的区域。

- **1x1 卷积**是用 1x1 滤波器做的卷积。它不关注空间邻域，而是在通道间混合信息。可以把它看作在每个空间位置上应用一个稠密层。它被用来廉价地改变通道数。

- **跳跃连接（skip connections，残差连接）**让输入绕过一层或多层：$\text{output} = F(x) + x$。该层只需学习残差 $F(x) = \text{output} - x$，当最优变换接近恒等映射时这更容易学。ResNet（残差网络）用这个技巧堆叠了超过 100 层，解决了更深的网络反而不如浅网络表现好的退化问题。

- CNN 构建 **特征层级**。早期层检测边缘和纹理，中间层将它们组合成部件（眼睛、轮子），后期层识别整个物体。每层的感受野（它能“看到”的输入区域）随深度增长。

- **嵌入（Embeddings）**将离散 token（词、字符、物品 ID）映射为稠密向量。嵌入层就是一个查找表：一个形状为（词表大小, 嵌入维度）的矩阵 $E$。查找 token $i$ 就是选取 $E$ 的第 $i$ 行。这等价于乘以一个 one-hot 向量，是矩阵-向量乘法的一个特例（第 02 章）。嵌入在训练中学习，因此相似的 token 最终会得到相似的向量。

- **分词（Tokenisation）**是将原始文本转换为 token 序列的过程。词级分词按空格切分，但无法处理未登录词。**子词分词（subword tokenisation）**（BPE、WordPiece、SentencePiece）将文本拆分成频繁出现的子词单元，在词表大小和覆盖范围之间取得平衡。单词 "unhappiness" 可能变成 ["un", "happiness"] 或 ["un", "happ", "iness"]。

- **循环神经网络（Recurrent Neural Networks, RNN）**逐元素处理序列，维护一个向前传递信息的隐藏状态：

$$h_t = \tanh(W_h h_{t-1} + W_x x_t + b)$$

- 隐藏状态 $h_t$ 是网络截至时刻 $t$ 所见一切的压缩摘要。同样的权重 $W_h$ 和 $W_x$ 在所有时间步共享（权重共享，类似于 CNN 在空间上共享权重）。

- 原始 RNN 难以处理长序列，因为梯度消失：从步 $t$ 到步 $t - k$ 的梯度信号要经过 $k$ 次与 $W_h$ 的乘法，并指数级地缩小（或爆炸）。

- **LSTM**（Long Short-Term Memory）通过引入一个几乎不受干扰地随时间流动的独立单元状态 $c_t$ 来解决这一问题。三个门控制哪些信息进入、离开和保留：

- **遗忘门**决定从单元状态中擦除什么：$f_t = \sigma(W_f [h_{t-1}, x_t] + b_f)$
- **输入门**决定写入什么新信息：$i_t = \sigma(W_i [h_{t-1}, x_t] + b_i)$，候选值为 $\tilde{c}_t = \tanh(W_c [h_{t-1}, x_t] + b_c)$
- 单元状态更新：$c_t = f_t \odot c_{t-1} + i_t \odot \tilde{c}_t$
- **输出门**决定暴露什么：$o_t = \sigma(W_o [h_{t-1}, x_t] + b_o)$，且 $h_t = o_t \odot \tanh(c_t)$

![LSTM 单元，显示遗忘门、输入门、输出门、单元状态高速通路以及数据流连接](../images/rnn_lstm_cell.svg)

- 单元状态像传送带一样工作：信息可以跨越多个时间步几乎不变地流动（遗忘门接近 1），这解决了长程依赖的梯度消失问题。

- **GRU**（Gated Recurrent Unit）将单元状态和隐藏状态合并为一个，并用两个门替代三个：一个更新门（合并遗忘和输入）和一个重置门，从而简化了 LSTM。GRU 参数更少，性能常常与 LSTM 相当。

- RNN（包括 LSTM）的根本局限在于顺序处理：必须先处理 token 1 再处理 token 2 再处理 token 3。这阻碍了并行化，并造成信息瓶颈，因为所有上下文都必须挤过固定大小的隐藏状态。

- **Attention** 同时解决了这两个问题。与其把整个输入压缩成一个固定向量，attention 让模型回顾所有输入位置，并决定哪些对当前输出相关。

- 现代表述使用 **查询、键、值（queries, keys, values, Q、K、V）**。可以把它想成图书馆检索：你有一个 query（你在找什么）、keys（每本书上的标签）和 values（书的实际内容）。你将自己的 query 与所有 keys 比较以决定检索哪些 values。

- **缩放点积 attention**：

$$\text{Attention}(Q, K, V) = \text{softmax}\!\left(\frac{QK^T}{\sqrt{d_k}}\right) V$$

- $QK^T$ 计算每个 query 与每个 key 之间的相似度。这是一次矩阵乘法（第 02 章），其元素是点积，度量余弦相似度（第 01 章）。除以 $\sqrt{d_k}$ 防止点积过大（否则 softmax 会饱和并产生近似 one-hot、梯度消失的分布）。softmax 把相似度转换为概率分布。乘以 $V$ 得到 values 的加权组合。

- **多头注意力（Multi-head attention）**并行运行 $h$ 个 attention 运算，每个使用不同的、学习得到的 Q、K、V 投影。这使模型能同时关注来自不同表示子空间的信息。一个头可能关注句法关系，另一个关注语义关系。输出被拼接后再投影：

$$\text{MultiHead}(Q, K, V) = \text{Concat}(\text{head}_1, \ldots, \text{head}_h) W^O$$

- **Transformer** 架构（Vaswani et al., 2017）完全由 attention 和前馈层构成，没有循环。编码器块重复：多头自注意力、相加并层归一化、前馈网络、相加并层归一化。解码器块额外添加了带掩码的自注意力（防止模型看到未来 token）和一个对编码器输出进行 attention 的交叉注意力层。

![Transformer 编码器块：多头注意力、相加并 layernorm、前馈网络、相加并 layernorm，带有残差连接](../images/transformer_block.svg)

- **位置编码（Positional encoding）**是必要的，因为 attention 是置换等变的，即它把输入当作集合而非序列。如果没有位置信息，"the cat sat on the mat" 和 "the mat sat on the cat" 将完全相同。原始 Transformer 使用正弦位置编码：

$$PE_{(pos, 2i)} = \sin\!\left(\frac{pos}{10000^{2i/d}}\right), \quad PE_{(pos, 2i+1)} = \cos\!\left(\frac{pos}{10000^{2i/d}}\right)$$

- 每个位置得到一个唯一的向量，模型可借此区分位置。现代模型常使用可学习的位置嵌入或相对位置编码（RoPE、ALiBi）替代。

- Transformer 并行处理所有 token（自注意力矩阵 $QK^T$ 在一次矩阵乘法中算出），因此在现代硬件上训练比 RNN 快得多。代价是自注意力在序列长度上是 $O(n^2)$（每个 token 都关注其他所有 token），而 RNN 是 $O(n)$。这也是长上下文模型需要特殊 attention 变体（稀疏注意力、线性注意力、flash attention）的原因。

- **视觉 Transformer（Vision Transformers, ViT）**把 Transformer 应用于图像：将图像切成固定大小的 patch（如 16x16），把每个 patch 展平为向量，并将这些 patch 当作 token 序列。一个可学习的 [CLS] token 被加在最前，其最终表示用于分类。尽管没有卷积归纳偏置，ViT 在足够数据上训练时能匹敌甚至超越 CNN。

- **MLP-Mixer** 是更简单的架构，用 MLP 同时替代 attention 和卷积。它在 “token-mixing” MLP（跨空间位置应用）和 “channel-mixing” MLP（跨特征应用）之间交替。它表现具竞争力，暗示现代架构的关键洞见不在于 attention 本身，而在于跨 token 和特征高效地混合信息。

- **自编码器（Autoencoders）**通过训练网络重建自身输入来学习压缩表示。编码器把输入映射到较低维的瓶颈（latent code），解码器再把它映射回去：

$$z = f_{\text{enc}}(x), \quad \hat{x} = f_{\text{dec}}(z), \quad \mathcal{L} = \|x - \hat{x}\|^2$$

- 瓶颈迫使网络学习最重要的特征。自编码器用于降维、去噪（在带噪输入上训练，重建干净输出）和异常检测（高重建误差意味着异常输入）。

- **变分自编码器（Variational Autoencoders, VAE）**加入了概率转折。编码器不再编码为单个点 $z$，而是输出一个分布的参数（高斯分布的均值 $\mu$ 和方差 $\sigma^2$）。latent code 从该分布采样：$z = \mu + \sigma \odot \epsilon$，其中 $\epsilon \sim \mathcal{N}(0, I)$。这个 **重参数化技巧（reparameterisation trick）**使采样可微，梯度得以流过。

- VAE 的损失有两项：

$$\mathcal{L} = \underbrace{\|x - \hat{x}\|^2}_{\text{reconstruction}} + \underbrace{D_{\text{KL}}(q(z|x) \| p(z))}_{\text{regularisation}}$$

- KL 散度项（第 05 章）推动学到的后验 $q(z|x)$ 逼近先验 $p(z) = \mathcal{N}(0, I)$，保证 latent space 平滑且结构良好。随后可以从先验采样并解码来生成新数据。这正是 VAE 作为生成模型的原因。

## 编程任务（使用 CoLab 或 notebook）

1. 用 JAX 从头构建一个简单的 MLP。在 2D 分类问题（如同心圆）上训练它，并可视化决策边界。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt
from sklearn.datasets import make_circles

# Data
X, y = make_circles(n_samples=500, noise=0.1, factor=0.5, random_state=42)
X, y = jnp.array(X), jnp.array(y, dtype=jnp.float32)

# Initialise a 2-layer MLP: 2 -> 16 -> 16 -> 1
def init_params(key):
    k1, k2, k3 = jax.random.split(key, 3)
    return {
        'W1': jax.random.normal(k1, (2, 16)) * 0.5,
        'b1': jnp.zeros(16),
        'W2': jax.random.normal(k2, (16, 16)) * 0.5,
        'b2': jnp.zeros(16),
        'W3': jax.random.normal(k3, (16, 1)) * 0.5,
        'b3': jnp.zeros(1),
    }

def forward(params, x):
    h = jnp.maximum(0, x @ params['W1'] + params['b1'])  # ReLU
    h = jnp.maximum(0, h @ params['W2'] + params['b2'])   # ReLU
    logit = (h @ params['W3'] + params['b3']).squeeze()
    return jax.nn.sigmoid(logit)

def loss_fn(params, X, y):
    pred = forward(params, X)
    return -jnp.mean(y * jnp.log(pred + 1e-7) + (1 - y) * jnp.log(1 - pred + 1e-7))

grad_fn = jax.jit(jax.grad(loss_fn))
params = init_params(jax.random.PRNGKey(0))
lr = 0.1

for step in range(2000):
    grads = grad_fn(params, X, y)
    params = {k: params[k] - lr * grads[k] for k in params}

# Plot decision boundary
xx, yy = jnp.meshgrid(jnp.linspace(-2, 2, 200), jnp.linspace(-2, 2, 200))
grid = jnp.column_stack([xx.ravel(), yy.ravel()])
zz = forward(params, grid).reshape(xx.shape)

plt.figure(figsize=(7, 6))
plt.contourf(xx, yy, zz, levels=[0, 0.5, 1], alpha=0.3, colors=['#e74c3c', '#3498db'])
plt.scatter(X[y==0,0], X[y==0,1], c='#e74c3c', s=10, label='Class 0')
plt.scatter(X[y==1,0], X[y==1,1], c='#3498db', s=10, label='Class 1')
plt.title("MLP Decision Boundary on Concentric Circles")
plt.legend(); plt.grid(alpha=0.3); plt.show()

acc = jnp.mean((forward(params, X) > 0.5) == y)
print(f"Accuracy: {acc:.2%}")
```

2. 从头实现 1D 卷积。对一个信号应用简单的边缘检测滤波器，并与内置的 `jnp.convolve` 对比。
```python
import jax.numpy as jnp
import matplotlib.pyplot as plt

def conv1d(signal, kernel):
    """1D convolution (valid mode) from scratch."""
    n, k = len(signal), len(kernel)
    output = jnp.zeros(n - k + 1)
    for i in range(n - k + 1):
        output = output.at[i].set(jnp.sum(signal[i:i+k] * kernel))
    return output

# Create a signal with a step function
t = jnp.linspace(0, 4, 200)
signal = jnp.where(t < 1, 0.0, jnp.where(t < 2, 1.0, jnp.where(t < 3, 0.5, 1.5)))

# Edge detection kernel
edge_kernel = jnp.array([-1.0, 0.0, 1.0])

# Our implementation vs built-in
our_output = conv1d(signal, edge_kernel)
jnp_output = jnp.convolve(signal, edge_kernel, mode='valid')

fig, axes = plt.subplots(3, 1, figsize=(10, 6), sharex=True)
axes[0].plot(t, signal, color='#3498db', linewidth=1.5)
axes[0].set_title("Original Signal"); axes[0].set_ylabel("Value")

axes[1].plot(t[:len(our_output)], our_output, color='#e74c3c', linewidth=1.5)
axes[1].set_title("After Edge Detection (our conv1d)"); axes[1].set_ylabel("Value")

axes[2].plot(t[:len(jnp_output)], jnp_output, color='#27ae60', linewidth=1.5, linestyle='--')
axes[2].set_title("After Edge Detection (jnp.convolve)"); axes[2].set_ylabel("Value")
axes[2].set_xlabel("t")

plt.tight_layout(); plt.show()
print(f"Outputs match: {jnp.allclose(our_output, jnp_output)}")
```

3. 从头实现缩放点积 attention。计算一个小例子的注意力权重，并将注意力矩阵可视化为热图。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

def scaled_dot_product_attention(Q, K, V):
    """Scaled dot-product attention."""
    d_k = Q.shape[-1]
    scores = Q @ K.T / jnp.sqrt(d_k)
    weights = jax.nn.softmax(scores, axis=-1)
    output = weights @ V
    return output, weights

# Example: 4 tokens, embedding dim 8
key = jax.random.PRNGKey(42)
k1, k2, k3 = jax.random.split(key, 3)
seq_len, d_model = 4, 8

K = jax.random.normal(k2, (seq_len, d_model))
V = jax.random.normal(k3, (seq_len, d_model))

output, weights = scaled_dot_product_attention(Q, K, V)

print(f"Q shape: {Q.shape}")
print(f"Attention weights shape: {weights.shape}")
print(f"Output shape: {output.shape}")
print(f"\nAttention weights (rows sum to 1):")
```

4. 构建一个简单的自编码器，将 2D 数据通过 1D 瓶颈压缩并重建。可视化 latent space 和重建结果。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt
from sklearn.datasets import make_moons

# Data
X, _ = make_moons(n_samples=500, noise=0.05, random_state=42)
X = jnp.array(X)

# Autoencoder: 2 -> 8 -> 1 -> 8 -> 2
def init_ae(key):
    k1, k2, k3, k4 = jax.random.split(key, 4)
    return {
        'enc_W1': jax.random.normal(k1, (2, 8)) * 0.5, 'enc_b1': jnp.zeros(8),
        'enc_W2': jax.random.normal(k2, (8, 1)) * 0.5, 'enc_b2': jnp.zeros(1),
        'dec_W1': jax.random.normal(k3, (1, 8)) * 0.5, 'dec_b1': jnp.zeros(8),
        'dec_W2': jax.random.normal(k4, (8, 2)) * 0.5, 'dec_b2': jnp.zeros(2),
    }

def encode(p, x):
    h = jnp.tanh(x @ p['enc_W1'] + p['enc_b1'])
    return h @ p['enc_W2'] + p['enc_b2']

def decode(p, z):
    h = jnp.tanh(z @ p['dec_W1'] + p['dec_b1'])
    return h @ p['dec_W2'] + p['dec_b2']

def ae_loss(p, X):
    z = encode(p, X)
    X_hat = decode(p, z)
    return jnp.mean((X - X_hat) ** 2)

grad_fn = jax.jit(jax.grad(ae_loss))
params = init_ae(jax.random.PRNGKey(0))
lr = 0.01

for step in range(3000):
    grads = grad_fn(params, X)
    params = {k: params[k] - lr * grads[k] for k in params}

z = encode(params, X)
X_hat = decode(params, z)

fig, axes = plt.subplots(1, 2, figsize=(12, 5))
axes[0].scatter(X[:,0], X[:,1], c=z.squeeze(), cmap='viridis', s=10)
axes[0].set_title("Original Data (coloured by latent code)")
axes[1].scatter(X_hat[:,0], X_hat[:,1], c=z.squeeze(), cmap='viridis', s=10)
axes[1].set_title("Reconstruction from 1D bottleneck")
for ax in axes:
    ax.set_aspect('equal'); ax.grid(alpha=0.3)
plt.tight_layout(); plt.show()

print(f"Reconstruction MSE: {ae_loss(params, X):.4f}")
```
